    메모관리 자주묻는 질문

    메모작성

    메모관리 → 메모작성

    메모를 '비공개'로 설정하면 수신자 외에는 열람이 불가능한가요?
    메모 보고 작성 시 '과제카드'를 선택하지 않으면 어떤 영향을 미치나요?
    메모보고 시 수신자 외에 다른 사람도 열람할 수 있나요? 수신자만 볼 수 있도록 설정하려면 어떻게 해야 하나요?
    메모 작성 시 '대국민공개여부'를 '부분공개' 또는 '비공개'로 설정할 때의 차이점은 무엇인가요?
    메모 작성 시 최근수신자는 무엇인가요 몇 명까지 표시되나요?

    임시저장메모

    메모관리 → 임시저장메모

    대국민공개여부를 선택하지 않고 메모를 임시저장할 경우 문제가 발생하나요?

    보고한메모

    메모관리 → 보고한메모

    작성한 메모를 회수하고 싶은데, 어떤 경우에 회수가 불가능한가요?
    보고한 메모를 삭제하거나 다른 메일함으로 이동할 수 있나요?

    보고받은메모

    메모관리 → 보고받은메모

    관리자(부서장/학교장)가 특정 메모를 '주요 메모'로 설정하는 방법은 무엇인가요?
    수신된 메모에 의견을 작성하거나 추가 수신자를 지정하는 방법은 무엇인가요?

    메모서식관리

    메모관리 → 메모서식관리

    메모 서식은 누가 등록하고 관리할 수 있으며, 어떤 서식 종류가 있나요?
    메모 작성시 자주사용하는 서식의 내용(제목, 보고자, 작성일 등)이 자동으로 입력될 수 있나요?

    신규메모

    메모관리 → 신규메모

    아직 읽지 않은 신규 메모를 일괄적으로 확인 처리하는 방법은 무엇인가요?

    메모등록대장

    메모관리 → 메모등록대장

    메모등록대장의 순번이 잘못 표시되는 이유는 무엇인가요?
    메모등록대장에서 조회되는 메모의 기준은 무엇인가요?

    개인수신그룹관리(메모)

    서비스공통 → 개인설정 → 개인수신그룹관리(메모)

    개인수신그룹관리에서 설정한 메모 그룹의 '보고/참조' 값이 메모 작성 시 불러와지지 않는 문제가 있습니다. 어떻게 처리하나요?

    개인환경설정(메모)

    서비스공통 → 개인설정 → 개인환경설정(메모)

    메모 작성 시 본문 크기를 조절하는 방법이 있나요?
    메모 작성 시 자주 선택하는 수신자나, 과제카드 정보를 기본값으로 설정할 있나요?
    이 페이지가 도움이 되었나요?
