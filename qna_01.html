    문서관리 자주묻는 질문

    개인정보관리

    서비스공통 → 개인설정 → 개인정보관리

    겸임 권한으로 문서를 기안할 때 전화번호가 공란으로 나올 경우 어떻게 해결 할 수 있나요? (X)

    겸임 권한으로 기안 시 전화번호가 공란이면, 개인문서환경설정에서 사용자 기본정보(전화번호)를 직접 등록·수정해야 합니다. 겸임 계정은 본인 정보가 자동 반영되지 않으므로 사용자 환경설정에서 전화번호를 저장하면 해결됩니다.

    결재직위명 설정 후에도 기존 문서의 결재직위가 변경되지 않을 경우 어떻게 해야하나요? (X)

    결재직위명은 설정 이후에 기안하는 문서부터 적용되므로, 이미 생성된 기존 문서의 결재직위는 자동 변경되지 않습니다. 기존 문서의 경우에는 해당 문서에서 직접 결재선을 수정해야 합니다.

    겸임 권한으로 문서 결재 처리 시 결재직위 설정은 어떻게 해야 하나요? (X)

    겸임 권한으로 결재할 때는 겸임 부서의 결재직위를 개인문서환경설정에서 미리 지정해야 합니다. 지정하지 않으면 결재 시 직위가 비어 있거나 기본값으로 표시되므로, 반드시 겸임 계정 환경설정에서 결재직위를 선택·저장한 뒤 사용해야 합니다.

    기안 시 '결재직위명이 없는 결재자가 있습니다.' 문구와 함께 기안되지 않습니다. (X)

    이 오류는 결재선에 포함된 사용자 중 결재직위가 지정되지 않은 경우 발생합니다. 해결 방법은 개인문서환경설정에서 결재직위명을 등록하거나, 겸임 계정이라면 겸임 부서 직위명을 지정한 후 다시 기안해야 합니다.

    부재정보관리

    서비스공통 → 개인설정 → 부재정보관리

    사용자부재설정 시 설정할 수 있는 대리결재자 알림설정은 어떤 기능인가요?

    사용자부재설정의 대리결재자 알림설정은, 부재 중 지정된 대리자가 결재할 문서가 발생하면 알림으로 안내해 주는 기능입니다. 이를 통해 대리결재자가 즉시 문서를 확인하고 처리할 수 있습니다.

    나이스시스템에서 부재 설정을 했는데 K-에듀파인에서 부재설정이 되지 않습니다.

    나이스 부재설정은 K-에듀파인과 연동되지 않으므로 별도로 적용되지 않습니다. K-에듀파인에서는 업무관리 → 사용자부재설정 메뉴에서 직접 부재 및 대리자를 설정해야 합니다.

    개인수신그룹관리

    서비스공통 → 개인설정 → 개인수신그룹관리

    개인수신그룹 일괄등록 시 입력해야 하는 정보에 관해 알려주세요. (X)

    개인수신그룹을 일괄등록할 때는 엑셀 서식에 구분, 아이디, 이름을 필수로 입력해야 합니다. 작성 후 업로드하면 오류가 없는 경우 저장됩니다.

    공람수신그룹 설정하는 방법을 알려주세요. (X)

    공람수신그룹은 개인문서환경설정에서 그룹을 등록한 후 문서 기안 시 수신처 지정 화면에서 선택해 사용하는 방식입니다. 즉, 미리 그룹에 사용자들을 지정해두면 공람 수신자 선택을 반복 입력하지 않고 일괄 지정할 수 있습니다.

    나의결재선관리

    서비스공통 → 개인설정 → 나의결재선관리

    나의결재선 작성 방법에 대해 알려주세요. (X)

    나의결재선은 개인문서환경설정에서 결재자를 미리 등록해 두는 기능입니다. 메뉴에서 [나의결재선] → [추가]를 선택해 결재선 이름을 입력하고, 조직도에서 결재자를 순서대로 지정·저장하면 됩니다. 이후 문서 기안 시 저장한 결재선을 불러와 바로 적용할 수 있습니다.

    개인문서환경설정

    서비스공통 → 개인설정 → 개인문서환경설정

    문서관리카드 사용자 화면 넓게 쓰기를 사용하고 싶은데, 어디서 설정할 수 있나요?

    문서관리카드 사용자 화면 넓게 쓰기는 개인문서환경설정 메뉴에서 설정할 수 있습니다. 해당 옵션을 체크하면 문서관리카드 화면을 기본보다 넓게 사용할 수 있습니다.

    개인문서환경설정에서 설정한 항목(트리보기, 기본 탭 등)은 타결재권자에게도 적용되나요?

    개인문서환경설정에서 지정한 트리보기, 기본 탭 등은 본인 계정에만 적용되며, 다른 결재권자에게는 영향을 주지 않습니다.

    문서 접수 시 업무관리자와 업무담당자의 결재 순서 기본값은 어떻게 설정하나요?

    문서 접수 시 결재 순서 기본값은 개인문서환경설정에서 ‘업무관리자 → 업무담당자’ 순서로 지정할 수 있습니다. 설정 후에는 새로 접수되는 문서에 해당 순서가 자동 반영됩니다.

    개인문서환경설정의 기본과제카드 설정이 접수문서에도 적용되나요?

    개인문서환경설정에서 지정한 기본과제카드는 기안 문서에만 적용되며, 접수문서에는 적용되지 않습니다.

    개인환경설정

    서비스공통 → 개인설정 → 개인환경설정

    알림존 발송대기에 숫자와 문서가 보이지 않습니다. (X)

    알림존 발송대기에서 숫자와 문서가 보이지 않는 경우는 개인문서환경설정에서 알림존 알림 항목을 미설정했기 때문입니다. 해당 메뉴에서 알림 항목을 지정해야 정상적으로 표시됩니다.

    문서환경설정

    서비스공통 → 포털관리 → 운영환경관리 → 문서환경설정

    문서 처리 시 개인정보필터가 적용되게 하려면 어떻게 해야 하나요? (X)

    문서 처리 시 개인정보필터를 적용하려면 개인문서환경설정에서 개인정보필터 사용 여부를 [사용]으로 설정해야 합니다. 이렇게 하면 기안·결재 과정에서 자동으로 개인정보 검출 기능이 작동합니다.

    조직관리

    서비스공통 → 조직및사용자 → (업무)조직관리 → 조직관리

    표준 서식의 상, 하단 홍보문구 또는 로고의 변경은 어디서 어떻게 할 수 있나요? (X)

    표준 서식의 상·하단 홍보문구나 로고는 기관 관리자 권한으로 업무관리 → 문서서식관리에서 수정할 수 있습니다. 해당 메뉴에서 서식을 선택 후 홍보문구·로고 항목을 변경하여 저장하면 적용됩니다.

    발송 전 발신명의 변경이 가능한가요? (X)

    발송 전에는 개인문서환경설정에서 발신명(기관명/부서명/성명 등) 항목을 선택해 변경할 수 있습니다. 지정한 발신명이 문서 발송 시 적용됩니다.

    사용자관리

    서비스공통 → 조직및사용자 → (업무)사용자관리 → 사용자관리

    문서 발송 시 '부서장이 설정되어있지 않습니다' 문구와 함께 발송처리가 되지 않습니다.

    이 오류는 발송 문서의 결재선에 부서장 직위가 지정되지 않은 경우 발생합니다. 해결하려면 조직도에서 해당 부서의 부서장을 먼저 지정한 뒤 문서를 다시 발송해야 합니다.

    조직도에 표기되는 사용자의 순서를 변경하려면 어떻게 해야 하나요? (X)

    조직도에 표기되는 사용자 순서는 개인문서환경설정의 조직도 사용자정렬 설정에서 변경할 수 있습니다. 여기서 정렬 기준(성명, 직위, 직급 등)을 선택해 저장하면 조직도에 반영됩니다.

    서비스공통 → 조직및사용자 → (업무)사용자관리 → 감사담당자관리

    외부사용자를 감사담당자로 지정하는 방법에 대해 상세히 알려주세요. (X)

    외부사용자는 기본 조직도에 없으므로, 감사담당자로 지정하려면 개인문서환경설정 → 외부사용자 등록에서 정보를 먼저 입력해야 합니다. 이후 문서 기안 시 결재선 또는 감사담당자 지정 화면에서 해당 외부사용자를 선택해 지정할 수 있습니다.

    접근권한관리

    서비스공통 → 권한관리 → (업무)접근권한관리 → 사용자별권한관리

    공문을 접수할 수 있는 권한을 사용자에게 설정하고 적용하려면 어떻게 해야 하나요? (X)

    공문 접수 권한은 기관관리자가 사용자 권한관리 메뉴에서 ‘공문접수권한’을 부여해야 합니다. 권한이 설정된 후부터 해당 사용자는 접수 메뉴에서 공문을 처리할 수 있습니다.

    기관장 권한을 부여하면 열람제한 문서를 포함한 기관내 모든 부서의 문서를 열람할 수 있나요?

    기관장 권한을 부여해도 열람제한 문서는 제외되며, 그 외 기관 내 모든 부서 문서는 열람할 수 있습니다.

    문서관리카드

    문서관리카드

    대국민공개여부 변경 시 대국민공개여부, 직원열람제한 설정 관련 팝업창을 나타나지 않게 할 수 없나요?

    대국민공개여부 변경 시 뜨는 공개·열람제한 설정 팝업은 필수 절차라서 숨기거나 생략할 수 없습니다. 반드시 해당 창에서 공개범위와 열람제한 여부를 확인·저장해야 합니다.

    문서관리카드 본문 하단 결재일자, 시행일자 관련 문의 드립니다.

    문서관리카드 본문 하단의 결재일자와 시행일자는 시스템에서 결재 및 발송 처리 시 자동으로 반영됩니다. 사용자가 직접 입력·수정할 수 없으며, 결재 완료·발송 완료 시점에 맞춰 표시됩니다.

    문서관리카드의 관련정보를 설정할 수 있는 방법을 알려주세요. (X)

    문서관리카드의 관련정보는 기안 화면에서 ‘관련정보’ 항목에 과제, 과제카드, 일정, 메모, 일지 등을 선택·연결하여 설정할 수 있습니다. 저장하면 해당 문서와 관련정보가 카드에 함께 표시됩니다.

    수신자숨김 기능이 적용 가능한 기준에 대해 알려주세요. (X)

    수신자숨김은 대외 발송 문서에서 여러 수신처를 지정했을 때, 수신처 간에 서로의 정보를 보이지 않도록 하는 기능입니다. 내부 수신이나 기관 내부 결재에서는 적용되지 않습니다.

    조직정보를 수정할 경우 문서번호가 어떻게 변경되는지 기준과 시점을 알고 싶어요.

    조직정보가 수정되면 이후에 기안·접수되는 문서부터 변경된 조직명이 문서번호에 반영됩니다. 이미 생성된 문서의 번호는 소급 변경되지 않습니다.

    공용서식

    문서관리 → 기안 → 공용서식

    현재 서식의 자동저장문서가 존재합니다. (X)

    이는 작성 중 문서가 자동저장 상태로 남아있다는 안내입니다. 문서 작성 화면에서 자동저장목록을 불러와 이어서 작성하거나, 불필요하면 삭제해야 새 문서 작성이 가능합니다.

    문서 기안 시 사용가능한 일괄기안은 어떤 기능인가요? (X)

    일괄기안은 여러 건의 문서를 동일한 서식과 결재선으로 한 번에 작성·기안할 수 있는 기능입니다. 반복적인 문서를 개별 작성하지 않고 엑셀 업로드 방식으로 일괄 처리할 수 있습니다.

    반려한 품의기안 삭제 요청 드립니다. (X)

    반려된 품의기안은 작성자가 직접 삭제할 수 없습니다. 문서관리자가 해당 문서를 확인 후 삭제 처리해야 하므로, 관리자에게 삭제 요청을 해야 합니다.

    지출결의 문서의 대국민공개여부와 문서관리 연계 재정문서의 대국민공개여부 설정관계에 대해 알려주세요. (X)

    지출결의 문서의 대국민공개여부는 재정시스템 연계 문서와 동일하게 적용됩니다. 즉, 지출결의에서 비공개로 설정하면 연계된 재정문서도 비공개 처리되며, 공개 설정 시에도 동일하게 연동됩니다.

    열람시암호확인이 설정된 문서는 일괄결재가 안되는것일까요?

    맞습니다. 열람 시 암호확인이 설정된 문서는 보안 제한 때문에 일괄결재 대상에서 제외됩니다. 해당 문서는 반드시 개별 결재로 처리해야 합니다.

    추가재발송요청 승인된 문서의 발송은 어떻게 하나요? (X)

    추가재발송요청이 승인된 문서는 문서관리카드에서 [발송] 버튼을 눌러 재발송하면 됩니다. 승인 상태가 되어야 발송 기능이 활성화됩니다.

    추가재발송요청을 취소하려면 어떻게 해야 하나요? (X)

    추가재발송요청은 문서관리카드에서 해당 요청을 선택 후 [요청취소] 버튼을 눌러 취소할 수 있습니다. 취소하면 문서는 원래 상태로 되돌아가며, 발송도 진행되지 않습니다.

    공문게시

    공문게시

    발송처리한 문서를 공문게시로 발송변경요청 할 수 있나요? (X)

    네, 가능합니다. 발송처리한 문서는 문서관리카드에서 ‘발송변경요청’을 통해 공문게시로 전환 요청할 수 있으며, 승인되면 발송방식이 공문게시로 변경됩니다.

    문서발송 시 설정할 수 있는 공문게시는 어떤 기능인가요?

    공문게시는 문서를 개별 수신처로 보내지 않고 게시판에 등록해 열람하도록 하는 발송 방식입니다. 지정된 기관이나 학교는 해당 게시판을 통해 공문을 확인할 수 있습니다.

    공문게시 주제영역은 공통사항인가요?

    공문게시 주제영역은 기관에서 공통으로 사용하는 분류 체계이며, 게시된 공문을 주제별로 검색·관리할 수 있도록 제공됩니다.

    접수대기의 일괄접수공문게시는 어떤 기능인가요?

    접수대기의 일괄접수공문게시는 여러 건의 공문게시 문서를 한 번에 선택해 접수 처리하는 기능입니다. 개별 문서마다 접수하지 않고 일괄로 처리해 업무를 단축할 수 있습니다.

    공람

    공람

    공람수신그룹을 통해 공람을 지정할 때 공람 지정이 되지 않는 경우에는 어떤 것이 있나요? (X)

    공람수신그룹을 지정해도 진행 중인 문서나 보안문서는 공람 대상으로 지정되지 않습니다. 따라서 이런 경우에는 공람이 적용되지 않습니다.

    공람 재지정을 하려면 어떻게 해야 하나요?

    공람 재지정은 문서관리카드 → 공람지정 화면에서 기존 공람자를 해제 후 새로운 공람자를 선택·저장하면 됩니다. 이렇게 하면 문서에 대한 공람 대상이 변경됩니다.

    공람 지정 후 공람을 회수 할 수 있는 방법에는 어떤게 있을까요? (X)

    공람 지정 후에는 문서관리카드의 공람관리 메뉴에서 회수할 공람자를 선택해 [공람해제] 처리해야 합니다. 해제하면 해당 사용자에게서 공람 권한이 사라집니다.

    직원열람제한 설정이 되어 있는 문서를 공람하려면 어떻게 해야 하나요? (X)

    직원열람제한이 설정된 문서는 제한 대상자를 제외한 사용자만 공람 지정 가능합니다. 따라서 제한이 걸린 직원은 공람 수신자로 지정할 수 없습니다.

    발송정보

    발송정보

    대외시행 문서로 기안한 문서를 결재 완료 후 수신자를 삭제할 수 있나요? (X)

    대외시행 문서는 결재 완료 후에는 수신자 삭제가 불가합니다. 수신처 변경이 필요하다면 발송변경요청 절차를 통해 처리해야 합니다.

    동일 수신처로 재발송한 건은 취소할 수 없나요? (X)

    맞습니다. 동일 수신처로 이미 재발송한 건은 취소할 수 없으며, 추가 변경이나 취소가 필요하면 발송변경요청을 통해 처리해야 합니다.

    발송한 공문의 수신여부를 확인할 수 있나요? (X)

    네. 발송한 공문의 수신여부는 문서관리카드의 발송이력 조회 기능에서 확인할 수 있습니다. 여기서 수신처별 수신 여부와 일시가 표시됩니다.

    발송대기

    문서관리 → 발송함 → 발송대기

    기안자만 문서를 발송할 수 있나요? (X)

    아닙니다. 기안자 외에도 문서관리자가 권한을 부여하면 다른 사용자도 발송할 수 있습니다. 다만 권한이 없는 경우에는 기안자만 발송 가능합니다.

    발송대기함의 문서를 발송하지 않으려면 어떻게 처리해야 하나요? (X)

    발송대기함의 문서를 발송하지 않으려면 문서관리카드에서 [발송취소] 처리해야 합니다. 취소하면 해당 문서는 발송되지 않고 대기함에서도 제외됩니다.

    발송처리

    문서관리 → 발송함 → 발송처리

    비전자발송은 문서 기안 시 발송유형을 ‘비전자’로 선택하고, 수신처에 이메일 주소나 일반 FAX 번호를 입력해 처리합니다. 이후 [발송] 시 해당 방식으로 전송됩니다.

    수신처로 문서24 수신 기관과 수기입력 수신 기관을 함께 동시 발송처리 할 수 있나요?

    네. 수신처 선택 시 문서24 연계 기관과 수기입력한 기관을 함께 지정할 수 있으며, 발송 시 동시에 처리됩니다.

    배부대기

    배부대기함에서 배부한 문서는 배부취소 기능으로 회수할 수 있습니다. 해당 문서를 선택 후 [배부취소] 처리하면 문서가 다시 배부 전 상태로 돌아갑니다.

    접수대기

    문서관리 → 접수함 → 접수대기

    접수대기의 알림존을 클릭하면 진행중인 문서가 조회되지 않습니다. 왜 그런가요? (X)

    접수대기 알림존에서 문서가 조회되지 않는 경우는 개인문서환경설정에서 알림존 항목을 미설정했거나, 이미 다른 사용자가 접수 처리해 대기 건이 없는 경우입니다.

    업무담당자를 지정하지 않고 접수 처리만 된 문서에 담당자를 지정하려면 어떻게 해야 하나요? (X)

    접수 처리만 된 문서는 문서관리카드 상세화면에서 [업무담당자 지정] 기능을 이용해 담당자를 추가할 수 있습니다. 저장 후부터 해당 담당자가 문서를 관리할 수 있습니다.

    접수한 문서의 결재경로를 변경하는 방법에 대해 알려주세요.

    접수한 문서의 결재경로는 문서관리카드에서 결재선 수정 기능을 통해 변경할 수 있습니다. 결재선 탭에서 기존 결재자를 삭제하거나 추가 지정한 후 저장하면 새로운 경로가 적용됩니다.

    접수대기함 문서 중 재발송요청 상태의 문서를 접수 할 수 없는데 어떻게 처리해야 하나요? (X)

    재발송요청 상태의 문서는 발송 변경 절차가 완료된 후에만 접수 가능합니다. 따라서 발송승인이나 재발송 처리가 끝난 뒤 접수해야 합니다.

    문서관리 → 접수함 → 업무당당확인

    업무담당확인요청 후 재지정요청된 문서는 문서관리카드에서 새로 지정된 업무담당자를 선택해 저장해야 합니다. 지정이 완료되어야 해당 문서가 정상 처리됩니다.

    보안공유

    보안공유

    보안공유와 공람의 차이가 무엇인가요?

    보안공유는 지정한 사용자에게만 열람 권한을 제한적으로 부여하는 것이고, 공람은 공식적으로 열람 대상자를 지정해 문서를 공유하는 것입니다. 보안공유는 선택된 사용자 외에는 열람 불가하며, 공람은 지정된 공람수신자가 문서를 확인할 수 있습니다.

    보안공유를 하려면 어떤 권한이 필요한가요? (X)

    보안공유는 과제카드의 실적을 관리할 수 있는 권한(과제관리자 또는 과제담당자 권한 이상)이 있어야 설정할 수 있습니다.

    과제카드 포함 인수인계를 완료해도 열람할 수 없는 문서가 있나요? (X)

    네. 인수인계가 완료되더라도 영구열람제한 문서는 열람할 수 없습니다.

    보안공유 가능한 문서는 어떤 문서이며 보안공유문서만 모아놓은 메뉴가 따로 있나요? (X)

    보안공유는 진행 중이 아닌 일반 문서에만 지정할 수 있으며, 보안공유로 설정된 문서는 별도 메뉴가 아닌 문서관리카드 내 보안공유 탭에서만 조회할 수 있습니다.

    문서등록대장

    문서관리 → 문서함 → 문서등록대장

    제목을 마스킹 처리한 문서의 경우 정보공개포털의 정보목록에 어떻게 연계되는지 궁금합니다.

    제목을 마스킹 처리한 문서는 정보공개포털 연계 시에도 동일하게 마스킹된 제목으로 정보목록에 반영됩니다. 원본 제목은 공개되지 않습니다.

    제목마스킹 기능 사용을 하려면 어떻게 해야하나요? (X)

    제목마스킹은 문서 기안 시 제목 입력란에서 [제목마스킹] 옵션을 선택하면 적용됩니다. 저장하면 해당 문서의 제목이 마스킹 처리되어 표시·공개됩니다.

    문서등록대장에서 수신자 숨김 기능이 적용 가능한 기준에 대해 알려주세요. (X)

    문서등록대장에서 수신자 숨김은 대외발송 문서에 여러 수신처가 있을 때 수신처 간 정보를 보이지 않게 할 경우에만 적용됩니다. 내부문서나 기관 내부 결재에는 적용되지 않습니다.

    문서등록대장의 제목마스킹, 수신자숨김 처리를 취소할 수 있는 기능이 있나요? (X)

    네. 문서등록대장에서 제목마스킹과 수신자숨김은 설정 후에도 문서관리카드에서 해제(취소)할 수 있는 기능이 제공됩니다.

    결재선 포함 사용자 외에 보안공유를 할 수 없도록 제한할 수 있나요? (X)

    네. 보안공유는 기관 정책에 따라 결재선 포함 사용자로만 제한할 수 있습니다. 이 경우 결재선 외 사용자에게는 보안공유 지정이 불가합니다.

    전임자가 설정한 열람제한문서를 열람할 수 있는 방법이 있나요? (X)

    전임자가 설정한 열람제한문서는 설정된 제한이 해제되지 않는 한 후임자도 열람할 수 없습니다. 열람이 필요하면 문서관리자에게 제한 해제를 요청해야 합니다.

    부서장 권한의 사용자가 하위 부서의 모든 문서를 열람할 수 있나요? (X)

    부서장 권한은 자신이 소속된 부서 문서만 열람 가능하며, 하위 부서 문서는 열람할 수 없습니다.

    열람제한문서의 열람이력을 확인할 수 있나요? (X)

    네. 열람제한문서도 문서관리카드의 열람이력 조회 기능을 통해 누가 열람했는지 확인할 수 있습니다.

    문서등록대장 하단 공람정보의 사후보고 상태에 대해 알려주세요. (X)

    문서등록대장 하단 공람정보의 사후보고 상태는 공람 지정된 사용자가 문서를 열람한 뒤 사후보고로 처리했음을 표시하는 상태값입니다.

    문서의 대국민공개여부 항목이 자동으로 변경되는 이유는 무엇인가요? (X)

    문서의 대국민공개여부는 연계된 과제카드의 공개 설정을 따라 자동 변경됩니다. 즉, 과제카드에서 비공개·부분공개로 설정하면 문서도 동일하게 반영됩니다.

    결재완료된 문서의 대국민공개여부, 직원열람제한 항목을 수정하려면 어떻게 해야 하나요? (X)

    결재완료된 문서의 대국민공개여부나 직원열람제한은 문서관리카드에서 문서정보 수정 권한(문서관리자 권한 이상)이 있을 때만 변경할 수 있습니다.

    폐지학교 문서의 대국민공개여부 수정 방법 문의드립니다. (X)

    폐지학교 문서는 해당 기관에서 수정할 수 없으므로, 관할 교육청 문서관리자가 대국민공개여부를 대신 변경해야 합니다.

    문서처리시 작성한 의견의 공개여부와 정보공개포털 연계여부에 대해 알려주세요. (X)

    문서 처리 시 작성한 의견은 내부 열람용으로만 사용되며 대국민공개 대상이 아니므로 정보공개포털과 연계되지 않습니다.

    결재완료되어 문서등록대장에 등록된 문서를 삭제할 수 있나요? (X)

    아니요. 결재완료되어 문서등록대장에 등록된 문서는 삭제할 수 없으며, 필요 시 문서관리자가 보존기간 보정이나 별도 이관 절차로만 처리할 수 있습니다.

    첨부물분리등록한 내용을 삭제할 수 있나요? (X)

    첨부물분리등록은 문서 본문과 첨부파일을 구분해 기록으로 남기는 기능으로, 이미 등록된 분리첨부는 삭제할 수 없습니다. 필요 시 문서관리자가 기록물 정리 단계에서 보정 처리만 가능합니다.

    첨부파일 다운로드 시 생성되는 파일명의 기준에 대해 알려주세요. (X)

    첨부파일 다운로드 시 생성되는 파일명은 문서관리카드에 등록된 첨부물의 원본 파일명을 기준으로 하며, 동일 파일명이 존재할 경우 시스템이 일련번호를 자동 부여해 중복을 구분합니다.

    결재가 완료된 문서의 결재취소가 가능한가요? (X)

    결재가 완료된 문서는 사용자가 임의로 결재취소할 수 없으며, 결재취소가 필요하면 문서관리자 권한에서 별도 보정 절차로 처리해야 합니다.

    이송, 반송, 경유, 회신, 재작성이 가능한 문서의 기준을 알려주세요. (X)

    이송, 반송, 경유, 회신, 재작성은 결재가 완료되지 않고 진행 중인 문서에 한해 가능합니다. 이미 결재가 완료된 문서는 해당 기능을 사용할 수 없습니다.

    이송 문서 기안 시 용량초과 알림창으로 인해 기안되지 않는 경우, 어떻게 해결해야 하나요?

    이송 문서 기안 시 용량초과로 기안되지 않는 경우는 첨부파일 용량 제한을 초과했기 때문입니다. 해결하려면 불필요한 첨부를 삭제하거나, 파일을 분할·압축해 제한 용량 이하로 줄여야 합니다.

    수신문서의 반송 처리 방법에 대해 알려주세요. 또 반송처리하면 문서등록대장에서 사라지나요?

    수신문서는 문서관리카드에서 [반송] 버튼을 선택해 사유 입력 후 처리할 수 있습니다. 다만 반송 처리된 문서는 문서등록대장에서 삭제되지 않고, 상태값이 ‘반송’으로 표시되어 이력 관리가 가능합니다.

    문서 시행정보에 작성하는 경유 항목의 작성 및 처리 방법에 대해 알려주세요. (X)

    문서 시행정보의 경유 항목은 발송 전 단계에서 경유 부서를 지정하는 기능입니다. 기안 시 시행정보 화면에서 경유 부서를 입력하면, 지정된 경유 부서가 결재선 처리 후 해당 문서를 확인·처리할 수 있습니다.

    이미 종료된 문서의 담당자를 재지정 하고 싶습니다. 가능한가요? (X)

    이미 종료된 문서는 담당자 재지정이 불가능합니다. 필요 시 문서관리자가 별도 보정 처리로만 담당자 변경을 반영할 수 있습니다.

    비전자문서

    비전자문서

    과년도 비전자문서 소급등록 기간은 어떻게 정해지나요?

    과년도 비전자문서 소급등록 기간은 시도 기록물 담당자가 기록관 환경설정에서 지정하며, 최대 5월 31일까지로 제한됩니다. 이 기간을 넘기면 소급등록은 불가합니다.

    과년도 소급등록문서가 당년도 문서등록대장에서 검색되는게 맞나요? (X)

    아니요. 과년도 소급등록문서는 해당 생산연도의 문서등록대장에만 반영되며, 당년도 등록대장에서는 검색되지 않습니다.

    2년 이상 지난 비전자 문서를 소급 등록 할 수 있나요?

    비전자 문서의 소급등록은 시도에서 지정한 소급일자 내에서만 가능하며, 일반적으로 최대 전년도까지만 허용됩니다. 따라서 2년 이상 지난 문서는 소급등록할 수 없습니다.

    문서등록대장에서 비전자문서만 조회할 수 있는 방법이 있나요?

    네. 문서등록대장에서 검색 조건을 ‘비전자문서’로 설정하면 비전자문서만 조회할 수 있습니다.

    열람제한 설정된 비전자문서를 처리과문서담당자 권한으로 수정할 수 있나요? (X)

    아니요. 열람제한이 설정된 비전자문서는 처리과문서담당자 권한으로도 수정할 수 없으며, 문서관리자 권한에서만 제한 해제 후 수정이 가능합니다.

    비전자문서(생산)의 결재자를 변경하려면 어떻게 해야 하나요?

    비전자문서(생산)의 결재자를 변경하려면 문서등록대장에서 해당 문서를 선택 후 결재선 수정 기능을 이용해야 합니다. 변경 후 저장하면 새로운 결재자가 반영됩니다.

    결재완료된 비전자 문서를 삭제할 수 있나요? (X)

    아니요. 결재완료된 비전자 문서는 삭제할 수 없으며, 필요 시 문서관리자가 보존기간 보정이나 기록물이관 과정에서만 조정할 수 있습니다.

    미완료문서

    문서관리 → 문서함 → 미완료문서

    인사발령 이전 학교의 미완료문서 처리방법 문의드립니다. (X)

    인사발령 이전 학교의 미완료문서는 해당 학교 문서담당자가 완료 처리해야 합니다. 본인은 직접 처리할 수 없으며, 필요 시 관할 교육청 문서관리자에게 완료 요청을 해야 합니다.

    폐교의 미접수 문서에 대한 처리 방법 문의 드립니다. (X)

    폐교의 미접수 문서는 관할 교육청의 기록물담당자가 이관대행조직을 지정하여 접수·처리해야 합니다.

    통폐합된 학교의 미완료문서를 처리하는 방법에 대해 알려주세요. (X)

    통폐합된 학교의 미완료문서는 해당 학교의 나이스코드가 그대로 유지된 상태라면, 승계 학교에서 미완료문서를 완료 처리해야 합니다. 나이스코드가 변경된 경우에는 관할 교육청 기록물담당자가 대행 조직을 지정해 처리합니다.

    진행상태인 접수문서의 담당자를 재지정하려면 어떻게 해야 하나요? (X)

    진행상태인 접수문서는 문서관리카드 상세화면에서 [업무담당자 재지정] 기능을 통해 담당자를 변경할 수 있습니다.

    검토자가 문서 결재 후 결재를 취소할 수 있는 방법이 있나요? (X)

    검토자가 결재한 문서는 본인이 직접 결재취소할 수 없으며, 취소가 필요할 경우 문서관리자가 보정 처리로만 결재 취소가 가능합니다.

    외부공문

    외부공문

    과제카드 변경 후 외부기관공문협조게시 기능이 작동하지 않아요. 어떻게 해야 할까요? (X)

    과제카드를 변경한 경우에는 문서관리카드에서 관련 과제카드를 다시 지정·저장해야 외부기관공문협조게시 기능이 정상 작동합니다.

    외부공문관리담당자 권한 부여 방법에 대해 알려주세요. (X)

    외부공문관리담당자 권한은 기관관리자가 권한관리 메뉴에서 해당 사용자에게 ‘외부공문관리담당자’ 권한을 추가 부여해야 합니다. 부여 후부터 외부공문 접수 및 관리 기능을 사용할 수 있습니다.

    외부공문관리담당자로 자동 접수된 문서의 담당자를 변경할 수 있나요? (X)

    네. 외부공문관리담당자로 자동 접수된 문서는 문서관리카드에서 [업무담당자 지정/변경] 기능을 통해 다른 담당자로 변경할 수 있습니다.

    외부공문환경설정의 키워드 설정 방법에 대해 알려주세요. (X)

    외부공문환경설정의 키워드는 기관관리자 메뉴에서 [외부공문환경설정] → [키워드 관리]에서 등록합니다. 키워드를 입력·저장하면, 이후 해당 키워드가 포함된 외부공문은 자동으로 분류·접수됩니다.

    학교의 외부공문관리담당자 권한 사용자가 여러명일 경우, 어떤 기준으로 접수되나요? (X)

    학교에 외부공문관리담당자가 여러 명 지정된 경우, 최초로 로그인한 담당자 계정으로 자동 접수가 이루어집니다.
