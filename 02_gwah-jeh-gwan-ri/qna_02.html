    과제관리 자주묻는 질문

    과제카드함

    과제관리 → 과제카드관리 → 과제카드함

    이미 종료된 과제카드를 삭제하거나 다시 사용하고 싶은데 가능한가요?

    종료된 과제카드에 실적이 존재하지 않을 경우에만 삭제 가능합니다.
    시도 또는 부서 과제카드 담당자가 과제카드함 화면에서 종료된 과제카드의 종료승인을 취소합니다. 종료요청자는 나의과제 또는 부서과제 화면에서 종료요청취소를 진행하면 과제카드 종료가 취소 완료됩니다.

    종료된 과제카드의 이름을 새 과제카드에 사용하려고 하는데, 중복으로 인해 등록이 불가능합니다. 어떻게 해야 하나요?

    과제카드 등록시 과제카드명을 중복 체크하며, 이때 종료된 과제카드도 포함됩니다. 종료된 과제카드의 이름을 '[종료] 2024 과제명'처럼 바꿔 중복을 피하고, 이후 새 과제카드에 원래 이름을 사용합니다.
    만약, 종료된 과제카드의 이름변경이 어려운 경우 새 과제카드 이름을 '과제명(2025)' 또는 '과제명_신규' 등으로 구분자를 덧붙여 우선 등록할 수 있습니다.

    신규 과제카드 등록 시 '과제카드를 사용하지 않도록 설정된 부서는 등록 및 관리기능 사용 불가' 메시지가 반복적으로 뜨는 이유는?

    과제카드 사용여부는 시스템 운영 설정중 조직정보에 해당합니다. 소속 기관의 시스템 관리자 또는 K-에듀파인 시도 총괄 담당자에게 문의하여 설정을 변경하시기 바랍니다.

    조직개편으로 기관 명칭이 변경되었고, 기록물 이관까지 완료된 과제카드의 명칭도 변경하고 싶은데 가능한가요?

    기록물이관이 완료된 과제카드의 명칭은 변경 불가능합니다.

    비슷한 과제카드명이 존재하는데, 하나를 삭제하고 싶어요.

    실적이 존재하지 않는 과제카드는 삭제 가능합니다. 만약, 실적이 존재할 경우 사용하려는 과제카드로 모든 실적을 옮긴 후, 사용하지 않는 과제카드를 삭제하시면 됩니다. 실적 이동은 과제관리 > 과제카드관리 > 과제카드변경 화면에서 진행하시면 됩니다. 또한, 과제카드함 화면의 과제카드 상세팝업에서 실적관리 탭 또는 접수관리 탭의 과제카드변경 기능도 이용하실 수 있습니다.

    과제카드가 종료된 상태라 종료된 과제카드를 다시 사용하려고 했으나, "종료된 단위과제로 등록된 단위과제카드가 포함되어 있어 종료승인취소할 수 없습니다." 라고 뜹니다. 과제카드를 다시 사용하려면 어떻게 하면 될까요?

    종료된 단위과제에 등록된 과제카드는 종료승인취소가 불가능합니다. 단위과제는 보존기간 준칙변경 또는 시도별 관리 정책에 따라 교육부총괄 BRM(기능분류체계) 관리자만 종료 처리할 수 있습니다. 필요 시 시도 BRM 관리자에게 요청하시기 바랍니다.
    종료된 단위과제와 유사한 성격을 가진 단위과제에 과제카드를 신규로 등록하거나, 이미 등록된 다른 과제카드를 활용하시기 바랍니다.

    이전에 있었던 선생님의 과제카드 담당자 정보를 새로운 담당자에게 일괄로 등록할 수 있는 방법을 알려 주세요.

    과제카드함 화면에서 담당자 조건을 현재담당자 또는 이전담당자로 선택한 후 담당자 목록에서 현 담당자를 선택하고 조회합니다. 조회된 전체 과제카드를 선택한 후 담당자일괄처리 기능을 이용하여 새로운 담당자로 일괄 변경하시면 됩니다.

    학교 근무자의 경우, 과제카드 생성 및 등록이 가능한가요?

    • K-에듀파인 업무관리 시스템에서 학교 근무자도 과제카드를 생성 및 등록을 요청할 수 있습니다.
    • 과제카드 등록 및 승인 권한에 대한 상세 설명
      1. 학교의 등록 요청 권한
        • 학교의 부서/학교 과제카드 담당자 권한을 보유한 사용자는 과제카드 등록 가능합니다.
        • 담당자 역할: 부서/학교 과제카드 담당자는 업무 성격에 맞는 단위 과제를 선택하여 과제카드를 등록 요청할 수 있습니다.
      2. 등록 방법
        • 과제 카드 등록은 개별 등록 요청 외에도, 정해진 서식을 이용하여 파일을 업로드하는 방식으로 과제카드를 일괄 등록 요청할 수 있습니다.
      3. 최종 승인 권한 (학교 외 기관)
        • 학교에서 등록을 요청하더라도, 해당 과제카드를 업무에 사용할 수 있도록 확정하는 최종 승인 권한은 학교에 없습니다.
        • 승인 담당자: 등록 요청된 과제 카드는 과제카드 승인 담당자 또는 시도 과제카드 담당자의 권한을 가진 사용자가 과제카드 승인함에서 검토합니다.
        • 승인 주체: 등록 요청된 과제 카드는 관할 지역청이나 시도 과제카드 담당자 가 승인 또는 반려 처리 합니다.
      4. 업무 개시 시점
        • 과제카드 승인 담당자가 등록 요청 건을 승인한 후에야 바로 업무 추진이 가능해집니다.

    과제카드

    과제관리 → 과제카드관리 → 과제카드함 → 과제카드

    기록물 이관이 완료된 과제카드의 이름(명칭)을 변경할 수 있나요?

    K-에듀파인 시스템에서 기록물이관이 완료되면, 해당 기록물은 더 이상 수정이 불가능합니다.

    담당자일괄처리

    과제관리 → 과제카드관리 → 과제카드함 → 담당자일괄처리

    학기 초 담당자 변경으로 여러 과제카드의 담당자를 한 번에 등록하고 싶어요.

    과제카드함 화면에서 담당자일괄처리 기능을 제공합니다. 담당자변경이 필요한 과제카드 목록을 조회한 후 선택합니다. 담당자일괄처리 버튼을 클릭하여 열린 담당자일괄처리 팝업에서 관리자, 담당자, 내부관계자를 일괄 처리할 수 있습니다.
    과제카드에 과제관리자가 지정되지 않으면, 담당자 선택 목록에서 최상위에 선택된 사용자가 과제관리자로 자동 지정됩니다.

    과제담당자변경

    과제관리 → 과제카드관리 → 과제카드함 → 과제카드 → 과제담당자변경

    과제카드에 담당자를 지정하려는데 해당 선생님이 목록에 나타나지 않아요.

    K-에듀파인 시스템에서는 개인별로 사용자 검색여부를 설정할 수 있습니다. 만약 해당 선생님이 자신의 업무사용자 여부를 아니오로 설정해 놓았다면, 과제카드 담당자 지정 시 사용자 검색 목록에서 조회되지 않을 수 있습니다. 해당 설정은 개인설정 > 개인정보관리 화면의 업무사용자 항목에서 변경 가능합니다.

    과제카드 담당자를 여러 명 지정하는 방법과 과제관리자와 과제담당자의 차이는 무엇인가요?

    과제카드함에서 담당자를 변경할 과제카드를 선택합니다. 과제카드 상세팝업에서 과제담당자 변경 버튼을 클릭하여 담당자를 추가로 지정하실 수 있습니다.

    • 과제관리자: 과제카드 정보의 수정 권한을 보유한 관리자를 지칭하며, 과제카드당 한 명만 지정될 수 있습니다.
    • 과제담당자: 단위과제 또는 관리과제를 추진하는 업무 담당자를 지칭하며, 하나의 과제카드에 대하여 여러 명의 과제담당자를 지정할 수 있습니다.

    과제카드변경

    과제관리 → 과제카드관리 → 과제카드변경

    과제카드 실적을 변경하려고 하는데 "과제카드를 변경할 수 없습니다. 변경기간 및 변경여부를 확인바랍니다." 라고 뜹니다. 변경기간은 어떻게 수정해야하나요?

    담당자는 과제카드 변경기간 내에서만 실적을 변경할 수 있습니다. 과제카드 변경기간이 아니라면 시도총괄관리자에게 기간변경을 요청하셔야 합니다.

    과제카드승인함

    과제관리 → 과제카드관리 → 과제카드승인함

    과제카드 등록 시 승인 절차가 필요하다고 하는데, 누가 승인할 수 있으며, 학교는 직접 승인이 가능한가요?

    학교는 과제카드 승인 권한이 없습니다. 학교에서 요청한 과제카드는 관할 교육지원청 및 교육청의 과제카드승인담당자, 또는 시도과제카드담당자가 승인 가능합니다.

    • 과제카드승인담당자: 자신이 관할하는 기관에서 등록 요청한 과제카드를 승인하거나 반려할 수 있습니다.
    • 시도과제카드담당자: 모든 기관의 과제카드를 승인하거나 반려할 수 있습니다.

    신규 과제카드 등록 신청을 했는데 계속 요청 상태로 대기 중이에요. 어떻게 처리해야 하나요?

    • 요청기관: 해당 과제카드 승인 권한이 있는 과제카드승인담당자 또는 시도과제카드담당자에게 승인을 요청해야 합니다.
    • 승인담당자: 과제관리 > 과제카드관리 > 과제카드승인함 화면에서 신규 과제카드 등록 요청을 승인 또는 반려 처리합니다.

    과제카드인계

    과제관리 → 과제카드관리 → 과제카드인계

    인수인계된 과제카드는 어떻게 확인하며, 원래 담당자가 해당 과제카드를 계속 볼 수 있나요?

    인계자는 과제카드인계 화면, 인수자는 과제카드인수 화면에서 인계인수된 과제카드 목록 및 이관상태를 확인할 수 있습니다.
    원래 담당자는 해당 과제카드 열람이 불가능합니다. 인수 절차가 완료되면 기존 담당자(인계자)는 해당 과제카드의 실적 조회 및 편철 권한을 상실하게 됩니다.

    과제카드를 다른 부서로 인계인수할 때 유의할 점이 있나요?

    인계 등록시 부서내 종료된 과제카드만 조회되며, 해당 목록에서 인계할 과제카드를 선택해야 합니다.
    조직 변경 등의 이유로 기록물 이관이 완료된 처리과에 과제카드를 인계인수할 경우, 해당 처리과에 미이관 과제카드가 발생할 수 있습니다.

    예산이 연결된 과제카드를 인계인수할 때, 예산 이체를 먼저 해야 하나요?

    과제카드 인계인수와 예산이체는 별도의 작업으로 과제카드 인계인수시 예산이체를 강제하지 않습니다. 그러므로, 인계인수시 업무 참고용으로만 예산의 존재여부를 알려줍니다.
    다만, 예산 이체 없이 과제카드만 인계하면, 새 부서에서는 결의나 집행이 불가능합니다. 반대로, 예산 이체만 하고 과제카드를 인계하지 않으면 업무실적과 예산의 소관이 불일치되어 감사 시 문제될 수 있습니다.

    과제카드인수

    과제관리 → 과제카드관리 → 과제카드인수

    인수부서의 인수접수 상태 과제카드를 기존과제카드로 이관 버튼을 누르면 콤보박스에서 사전에 등록한 기존과제카드 명을 불러올 수 없다고 하는데 이유는?

    인수부서에 인수하려는 과제카드와 동일한 단위과제로 만들어진 과제카드가 존재하지 않으면 이관대상과제카드선택이 불가능합니다. 신규과제카드로 이관을 진행하거나, 이관 전 동일한 단위과제로 과제카드를 미리 생성하시기 바랍니다.

    나의과제

    과제관리 → 나의과제

    과제카드를 종료하려고 하는데 정리완료된 과제카드가 존재한다 또는 진행 중인 실적이 있어 종료가 안 된다는 메시지가 뜨는데 왜일까요?

    해당 과제카드를 포함한 처리과가 기록물이관 진행 전이라면, 기록물정리함 또는 기록물정리현황 화면에서 기록물 정리완료를 취소할 수 있습니다.
    진행중인 실적이 있어 종료가 안될 경우 문서관리 메뉴에서 해당 실적을 모두 완료하여야 과제카드 종료할 수 있습니다.

    조직개편 후 과제카드를 인수받았는데 목록에 보이지 않는데 이유는?

    과제카드 인계인수 절차가 완료되지 않거나, 인계받은 과제카드에 담당자로 지정되지 않으면 조회가 불가능합니다. 부서(학교) 과제카드 담당자에게 요청하여 과제카드 인계인수 완료 확인 및 담당자 권한 부여 요청하시기 바랍니다.
