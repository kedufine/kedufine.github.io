    과제관리 자주묻는 질문

    과제카드함

    과제관리 → 과제카드관리 → 과제카드함

    이미 종료된 과제카드를 삭제하거나 다시 사용하고 싶은데 가능한가요?
    종료된 과제카드의 이름을 새 과제카드에 사용하려고 하는데, 중복으로 인해 등록이 불가능합니다. 어떻게 해야 하나요?
    신규 과제카드 등록 시 '과제카드를 사용하지 않도록 설정된 부서는 등록 및 관리기능 사용 불가' 메시지가 반복적으로 뜹니다.
    조직개편으로 기관 명칭이 변경되었고, 기록물 이관까지 완료된 과제카드의 명칭도 변경하고 싶은데 가능한가요?
    비슷한 과제카드명이 존재하는데, 하나를 삭제하고 싶어요.
    과제카드가 종료된 상태라 종료된 과제카드를 다시 사용하려고 했으나, "종료된 단위과제로 등록된 단위과제카드가 포함되어 있어 종료승인취소할 수 없습니다." 라고 뜹니다. 과제카드를 다시 사용하려면 어떻게 하면 될까요?
    이전에 있었던 선생님의 과제카드 담당자 정보를 새로운 담당자에게 일괄로 등록할 수 있는 방법을 알려 주세요.

    과제카드

    과제관리 → 과제카드관리 → 과제카드함 → 과제카드

    기록물 이관이 완료된 과제카드의 이름(명칭)을 변경할 수 있나요?

    담당자일괄처리

    과제관리 → 과제카드관리 → 과제카드함 → 과제카드 → 담당자일괄처리

    학기 초 담당자 변경으로 여러 과제카드의 담당자를 한 번에 등록하고 싶어요.

    과제담당자변경

    과제관리 → 과제카드관리 → 과제카드함 → 과제카드 → 과제담당자변경

    과제카드에 담당자를 지정하려는데 해당 선생님이 목록에 나타나지 않아요.
    과제카드 담당자를 여러 명 지정하는 방법과 과제관리자와 과제담당자의 차이는 무엇인가요?

    과제카드변경

    과제관리 → 과제카드관리 → 과제카드변경

    과제카드 실적을 변경하려고 하는데 "과제카드를 변경할 수 없습니다. 변경기간 및 변경여부를 확인바랍니다." 라고 뜹니다. 변경기간은 어떻게 수정해야하나요?

    과제카드승인함

    과제관리 → 과제카드관리 → 과제카드승인함

    과제카드 등록 시 승인 절차가 필요하다고 하는데, 누가 승인할 수 있으며, 학교는 직접 승인이 가능한가요?
    신규 과제카드 등록 신청을 했는데 계속 요청 상태로 대기 중이에요. 어떻게 처리해야 하나요?

    과제카드인계

    과제관리 → 과제카드관리 → 과제카드인계

    인수인계된 과제카드는 어떻게 확인하며, 원래 담당자가 해당 과제카드를 계속 볼 수 있나요?
    과제카드를 다른 부서로 인계인수할 때 유의할 점이 있나요?
    예산이 연결된 과제카드를 인계인수할 때, 예산 이체를 먼저 해야 하나요?

    과제카드인수

    과제관리 → 과제카드관리 → 과제카드인수

    인수부서의 인수접수 상태 과제카드를 기존과제카드로 이관 버튼을 누르면 콤보박스에서 사전에 등록한 기존과제카드 명을 불러올 수 없습니다.

    나의과제

    과제관리 → 나의과제

    과제카드를 종료하려고 하는데 정리완료된 과제카드가 존재한다 또는 진행 중인 실적이 있어 종료가 안 된다는 메시지가 뜹니다.
    조직개편 후 과제카드를 인수받았는데 목록에 보이지 않습니다.
    이 페이지가 도움이 되었나요?
