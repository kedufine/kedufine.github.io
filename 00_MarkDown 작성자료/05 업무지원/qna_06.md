#업무지원 자주묻는 질문

## 최신게시물

:material-sitemap-outline: 업무지원 → 알림판 → 최신게시물

!!! question "게시판에 최근 등록된 게시물을 조회하는 방법은 무엇인가요?"
	게시판에서 최근 등록된 게시물은 게시판 목록에서 해당 게시판을 선택 후 ‘조회’ 버튼을 클릭하면 확인할 수 있으며, 제목을 클릭하면 상세 내용을 열람할 수 있습니다

## 공문게시판

:material-sitemap-outline: 업무지원 → 알림판 → 공문게시판

!!! question "공문게시판의 주제 영역 탭을 추가하거나 삭제할 수 있나요?"
	공문게시판의 주제영역은 서비스공통 > 코드관리 메뉴에서 공통코드로 관리됩니다.
	주제영역은 관리자가 공통코드에서 등록하거나 삭제할 수 있으며, 설정된 주제영역에 공문이 포함되어 있는 경우에만 메인화면에 탭이 표시되고 해당 공문을 조회할 수 있습니다.

!!! question "공문게시물을 조회하고 다운로드하는 방법은 무엇인가요?"
	공문게시물은 공문게시글 관리 메뉴에서 목록을 조회한 뒤, 제목을 클릭해 상세 화면을 열람할 수 있고, 필요한 경우 첨부파일을 다운로드하여 활용할 수 있습니다.

!!! question "게시글 등록 시 공문게시글에 등록될까요?"
	공문 게시판은 문서관리 시스템에서 공문 게시를 선택하여 만들어진 공문 게시물을 조회하는 화면입니다.
	문서 처리 시 공문 게시를 선택한 경우, 해당 공문이 공문 게시판에 실시간으로 조회됩니다.
	게시판에서 등록한 일반게시물은 공문게시글에 등록되지 않습니다.

## 일반게시판

:material-sitemap-outline: 업무지원 → 알림판 → 일반게시판

!!! question "게시판 관리에서 특정 게시판의 목록이 조회되지 않아 관리할 수 없습니다."
	게시판 목록이 조회되지 않는 경우는 해당 게시판이 **개설 취소 상태**이거나 **열람 범위에 포함되지 않았기 때문**입니다.
	따라서 게시판 관리 화면에서 해당 게시판을 선택 후, 개설 상태와 열람 범위 설정을 다시 확인·변경해야 합니다.

!!! question "학교에서 게시물 등록은 내부기안 후 등록을 하는건가요?"
	일반 게시판에 게시물을 등록하는 것은 별도의 공식적인 내부 기안 절차를 거치지 않습니다.

!!! question "게시물은 업무관리에 접속하는 모든 기관/학교가 보는걸까요?"
	- 게시글이 업무관리에 접속하는 모든 기관이나 학교에 자동으로 보이는 것은 아니며, 이는 **열람 범위** 설정에 따라 제한됩니다.
		1. 일반 게시판의 범위
			* 게시판은 관리자 권한을 보유한 사용자가 게시판을 개설해야 일반 사용자가 해당 게시판을 활용할 수 있게 되며, 게시판 등록 시 열람 범위가 설정됩니다.
			* 게시물 등록 시에도 게시판의 열람 범위 내에서만 열람 범위를 설정할 수 있습니다. 해당 게시판에 대한 접근 권한과 열람 범위 내에 있는 사용자만 게시글을 볼 수 있습니다.
		2. 공문 게시판의 범위
		    * 공문 게시판의 경우, 게시된 공문은 공문 작성 시 설정된 열람 기관 및 부서에 따라 그 범위가 제한됩니다.
			* 공문 게시물을 조회하는 화면에서 아이콘을 클릭하여 해당 게시물의 열람 기관 목록을 조회할 수 있습니다.
	- 알림판 구성
		1. 일반 게시판 : 일반 사용자가 직접 게시물 등록 권한에 따라 글을 작성하는 공간입니다.
		2. 공문 게시판 : 문서관리 시스템을 통해 결재된 공문이 자동으로 게시되는 공간입니다.
		3. 최신 게시물: 모든 게시판에 최근 등록된 게시물 목록을 조회하는 화면입니다.
		4. 자주 보는 게시물: 사용자가 개인적으로 중요하다고 선택하여 등록한 게시글 목록을 조회하는 화면입니다.

## 게시판관리

:material-sitemap-outline: 업무지원 → 알림판 → 일반게시판

!!! question "관리자가 없어진 일반게시판을 폐쇄(개설취소)하는 방법은 무엇인가요?"
	관리자가 없어진 일반게시판은 게시판 관리 메뉴에서 해당 게시판을 선택 후 **개설취소** 설정을 하면 폐쇄할 수 있습니다.

## 나의게시글관리

:material-sitemap-outline: 업무지원 → 알림판 → 공문게시판 → 나의게시글관리

!!! question "내가 작성한 공문게시물을 일괄 삭제하거나 정보를 변경하는 방법은 무엇인가요?"
	나의게시글관리 팝업에서 공문게시물 정보를 수정하거나, 조회된 공문을 선택하여 삭제 버튼을 클릭하면 삭제할 수 있습니다.
	공문게시물 수정 시에는 주제영역, 게시기간, 열람범위를 변경할 수 있습니다.


## 공고등록대장

:material-sitemap-outline: 업무지원 → 공고관리 → 공고등록대장

!!! question "공고등록대장의 일련번호를 수정할 수 없나요?"
	공고등록대장의 일련번호는 **자동 채번**되는 값으로 수정할 수 없으며, 필요 시 번호이력조회 기능을 통해 **소급 채번**만 가능합니다.

## 공고승인관리

:material-sitemap-outline: 업무지원 → 공고관리 → 공고승인관리

!!! question "공고 승인 후 게시 전에 등록 승인하면 통합 검색 시 모든 학교/기관에 노출되어 보안 문제가 발생하지 않을까요?"
	공고는 **승인 후 게시 전 단계에서는 통합검색에 노출되지 않으며**, 실제 게시가 완료되어야만 학교·기관에 공개됩니다.
	따라서 승인만으로 보안 문제가 발생하지는 않습니다.


## 공고대장설정

:material-sitemap-outline: 업무지원 → 공고관리 → 공고대장설정

!!! question "기관관리자(업무) 권한을 가졌는데 공고대장설정 메뉴가 보이지 않아요. 어떻게 처리해야 하나요?"
	기관관리자(업무) 권한이 있어도 공고대장설정 메뉴가 보이지 않는 경우는 기관관리자가 공고대장 신설 권한을 부여받지 않았거나 환경설정이 완료되지 않은 경우입니다.
	이때는 **시도총괄관리자에게 권한 및 메뉴 설정을 요청**해야 합니다.

## 홍보관리

:material-sitemap-outline: 업무지원 → 홍보관리 → 홍보요청

!!! question "학교 입학설명회를 홍보등록하려고 하면 홍보담당자는 학교의 공문접수담당자일까요?"
	홍보담당자는 교육지원청·시도교육청에 배정된 담당자이며, 학교의 공문접수담당자와는 역할이 다릅니다.
	따라서 입학설명회 홍보를 등록하려면 학교에서 홍보요청을 작성한 뒤, 교육지원청·시도교육청의 홍보담당자에게 접수 요청을 해야 합니다.
	
	- 홍보담당자의 역할 : 홍보담당자는 홍보 관련 환경을 설정하고, 일반 사용자가 요청한 홍보 건을 접수 및 처리하는 역할을 수행합니다.
	- 일반 사용자의 역할 : 일반 사용자는 홍보 요청 정보를 등록하고, 메모보고(메모작성)나 문서작성을 통해 홍보담당자에게 접수를 요청하는 기능만 수행합니다.

!!! question "요청등록을 하면 따로 지원청에서 승인 및 등록을 해주시는건지 궁금합니다."
	- 학교(일반 사용자)가 입학설명회 등의 홍보물을 등록하려고 요청하면, 교육지원청 또는 시도교육청의 홍보담당자가 최종적인 접수 및 등록 처리를 담당합니다.	
	- 홍보 요청 및 승인/등록 절차 (지원청 포함)
		  1. 홍보 요청 정보 등록 : 일반 사용자는 홍보 요청 화면에서 홍보 제목, 내용, 유형 등의 홍보 정보를 등록합니다.
		  2. 접수 요청 : 등록된 홍보 요청 건은 메모보고 또는 문서 작성 방식을 통해 홍보담당자에게 접수 요청 처리됩니다.
		  3. 요청 대상 기관 선택 : 홍보 요청 시, 교육지원청 소속 사용자는 교육지원청 또는 시도교육청 중 한 곳을 선택하여 요청할 수 있습니다. (시도교육청 사용자는 시도교육청 홍보담당자에게만 요청 가능합니다).
		  4. 접수 및 승인 처리 : 선택된 기관의 홍보담당자는 홍보접수 메뉴에서 요청된 홍보 건을 확인하고 접수(승인)하거나 반려 처리합니다.
		  5. 최종 등록 : 홍보담당자는 접수된 홍보 요청에 대해 홍보 처리 결과를 등록한 후, 최종적으로 **처리완료** 상태로 변경함으로써 홍보 등록을 확정하게 됩니다.
