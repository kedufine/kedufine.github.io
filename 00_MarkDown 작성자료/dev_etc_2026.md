# 2026년 문서관리 외 기능개선 내역

K-에듀파인 과제관리 등의 업무 기능개선 사항을 배포일자별로 세부사항을 제공합니다. 주요 기능 및 화면변경에 대해서만 작성하였고 오류수정 등은 포함하지 않습니다.

## 6월 23일

!!! success "[기록물이관] 기록물 사본 폐기 기능 적용"
	- 기록물 이관이 완료된 문서(사본)를 안전하게 폐기하고 관리할 수 있도록 '기록물이관마감관리' 화면과 '폐기진행현황' 화면을 신규 제공
	- 기록물이관마감관리
		1. 화면설명 : 기록물담당자가 이관 완료된 해당 연도의 미이관 기록물을 확인하고 이관 작업을 최종 마감하는 화면
		2. 화면경로 : 업무관리 > 과제관리 > 기록물이관 > 기록물이관마감관리
		3. 필요권한 : 시도기록물담당자
	- 폐기진행현황
		1. 화면설명 : 총괄관리자가 이관 마감된 생산년도를 대상으로 '조회제한'을 우선 적용한 후, 부서별 폐기 실적 수집 및 물리적 폐기 처리를 관리하는 화면
		2. 화면경로 : 업무관리 > 과제관리 > 기록물이관 > 폐기진행현황 
		3. 필요권한 : 시도총괄관리자(업무)
	- 폐기 작업 전 '조회제한' 설정시 사용자 화면에서 해당 폐기 연도의 문서 조회를 차단하는 기능 제공
??? success "[기록물이관] 대용량 별송파일 처리과 단위 다운로드 기능 추가"
    - 대용량 별송파일 다운로드를 전체 처리과 일괄실행에서 처리과 단위로 수행할 수 있도록 기능 개선
??? success "[일정관리] 일정관리 공개범위 세분화 및 하위 세부기관 업무일정 공유 기능 추가"
	- 교육청일정담당자 권한을 가진 사용자가 일정 작성시 공개범위를 교육청 전체에서 세분화 하여 설정할 수 있도록 개선
	- 공개범위 세분화 선택은 기관분류(시도교육청, 교육지원청, 직속기관, 학교)와 학교급(유치원, 초등학교, 중학교, 고등학교, 특수학교)별 제공
	- 공개범위를 교육청내공개로 선택시 열람 가능한 조직 확인을 위한 열람조직 조회 기능 제공

## 4월 28일

!!! success "[과제관리] 과제카드 그룹 관리 기능 제공"
	- 부서 내 다수의 과제카드를 업무 성격에 따라 그룹으로 구성하여 효율적으로 관리할 수 있는 기능 신규 제공
	- 업무 성격에 맞는 과제카드 그룹을 생성하여 과제카드를 그룹 단위로 관리할 수 있도록 지원
	- 화면경로 : 업무관리 > 과제관리 > 과제카드관리 > 과제카드그룹관리
	- 필요권한 : 부서·기관·시도 과제카드 담당자
	- 업무처리순서
		1. 과제카드 그룹 등록
		2. 등록된 그룹에 과제카드 일괄 등록
		3. 그룹 내 과제카드 담당자 일괄 지정
??? success "[기록물이관] 기록물이관 공휴일 설정 기능 추가"
    - 기록물이관 파일생성 작업을 공휴일에는 24시간 수행할 수 있도록 환경설정 기능 제공
