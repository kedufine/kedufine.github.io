#메모관리 자주묻는 질문

## 메모작성

:material-sitemap-outline: 메모관리 → 메모작성

!!! question "메모를 '비공개'로 설정하면 수신자 외에는 열람이 불가능한가요?"
    메모를 비공개 또는 부분공개로 설정하면 보고자 외 타 사용자에게는 열람되지 않으며, 지정된 수신자만 열람할 수 있음

!!! question "메모 보고 작성 시 '과제카드'를 선택하지 않으면 어떤 영향을 미치나요?"
	메모 보고 작성 시 과제카드를 선택하지 않으면 기록물이관 대상에서 제외되며, 과제카드를 선택한 경우에는 반드시 대국민 공개 여부를 설정해야 함

!!! question "메모보고 시 수신자 외에 다른 사람도 열람할 수 있나요? 수신자만 볼 수 있도록 설정하려면 어떻게 해야 하나요?"
    메모보고 시 기본적으로는 수신자가 열람할 수 있으며, 비공개 또는 부분공개로 설정하면 수신자 외 다른 사람은 열람할 수 없습니다. 따라서 수신자만 보도록 하려면 메모 작성 시 공개 설정을 비공개로 지정하여야 함.

!!! question "메모 작성 시 '대국민공개여부'를 '부분공개' 또는 '비공개'로 설정할 때의 차이점은 무엇인가요?"
    메모 작성 시 부분공개는 일부 내용만 제한적으로 공개하는 것이고, 비공개는 수신자 외에는 누구도 열람할 수 없도록 완전히 차단하는 설정임.

!!! question "메모 작성 시 최근수신자는 무엇인가요 몇 명까지 표시되나요?"
    메뉴얼에 따르면 메모 작성 시 최근수신자 기능은 사용자가 직전에 지정했던 수신자를 자동으로 불러와 편리하게 선택할 수 있는 기능이며, 최근 10명까지 표시

## 임시저장메모

:material-sitemap-outline: 메모관리 → 임시저장메모

!!! question "대국민공개여부를 선택하지 않고 메모를 임시저장할 경우 문제가 발생하나요?"
    과제카드를 선택한 경우에는 대국민공개여부 설정이 필수이므로 선택하지 않고 임시저장하면 정상 저장되지 않고 오류가 발생할 수 있음

## 보고한메모

:material-sitemap-outline: 메모관리 → 보고한메모

!!! question "작성한 메모를 회수하고 싶은데, 어떤 경우에 회수가 불가능한가요?"
    작성한 메모는 수신자가 열람했거나 의견이 등록된 경우에는 회수가 불가능하며, 수신자가 아직 열람하지 않았고 의견도 없는 상태일 때만 회수할 수 있음

!!! question "보고한 메모를 삭제하거나 다른 메일함으로 이동할 수 있나요?"
    보고한 메모는 이미 발송되어 삭제할 수 없으며, 회수된 메모만 임시저장메모에서 재작성·삭제 가능합니다. 또한 발신된 메모는 다른 메일함으로 이동할 수 없음


## 보고받은메모

:material-sitemap-outline: 메모관리 → 보고받은메모

!!! question "관리자(부서장/학교장)가 특정 메모를 '주요 메모'로 설정하는 방법은 무엇인가요?"
    관리자(부서장/학교장)는 확인 처리한 메모를 선택한 뒤 ‘관리자주요메모’ 버튼을 클릭하여 주요 메모로 설정할 수 있고, 해제할 때도 동일하게 체크 후 해제 버튼을 누르면 됩니다.

!!! question "수신된 메모에 의견을 작성하거나 추가 수신자를 지정하는 방법은 무엇인가요?"
    수신된 메모의 상세 화면에서 의견 내용을 작성 후 저장하면 의견 등록이 가능하며, 작성자가 수신자 추가를 허용한 경우에만 조직도나 검색 기능으로 수신자를 추가 지정할 수 있음

## 메모서식관리

:material-sitemap-outline: 메모관리 → 메모서식관리

!!! question "메모 서식은 누가 등록하고 관리할 수 있으며, 어떤 서식 종류가 있나요?"
    권한에 따라 등록 범위가 다름, 시도총괄관리자(개인·기관·공통 서식 등록), 기관관리자(개인·기관 서식 등록), 일반사용자(개인 서식만 등록)

!!! question "메모 작성시 자주사용하는 서식의 내용(제목, 보고자, 작성일 등)이 자동으로 입력될 수 있나요?"
    메모 작성 시 등록된 메모 서식을 선택하면 해당 서식의 제목, 보고자, 작성일 등 기본 내용이 본문에 자동 적용

## 신규메모

:material-sitemap-outline: 메모관리 → 신규메모

!!! question "아직 읽지 않은 신규 메모를 일괄적으로 확인 처리하는 방법은 무엇인가요?"
    신규메모 화면에서 열람이 필요한 메모를 여러 건 선택한 후, 화면 상단의 확인일괄처리 버튼을 클릭하면 선택한 메모를 한 번에 확인 처리할 수 있음

## 메모등록대장

:material-sitemap-outline: 메모관리 → 메모등록대장

!!! question "메모등록대장의 순번이 잘못 표시되는 이유는 무엇인가요?"
    메모등록대장은 보고일시를 기준으로 목록을 정렬하여 표시함. 따라서 메모가 새로 등록되거나 수정될 경우, 보고일시가 변경되어 순번이 바뀌어 보일 수 있음

!!! question "메모등록대장에서 조회되는 메모의 기준은 무엇인가요?"
    - 메모등록대장에서는 메모번호가 채번된 메모, 즉 보고가 완료된 메모만 조회할 수 있음
    - 조회 권한은 사용자 권한에 따라 다르며, 감사담당자, 처리과문서담당자, 부서·학교기록물담당자, 부서장 등은 부서 또는 기관 내에서 보고된 메모 조회 가능

## 개인수신그룹관리(메모)

:material-sitemap-outline: 서비스공통 → 개인설정 → 개인수신그룹관리(메모)

!!! question "개인수신그룹관리에서 설정한 메모 그룹의 '보고/참조' 값이 메모 작성 시 불러와지지 않는 문제가 있습니다. 어떻게 처리하나요?"
    메뉴얼에 따르면 개인수신그룹관리에서 설정한 ‘보고/참조’ 값이 메모 작성 시 불러와지지 않는 경우는 오류 상황이며, 이때는 메모작성 화면에서 수신자를 다시 직접 지정

## 개인환경설정(메모)

:material-sitemap-outline: 서비스공통 → 개인설정 → 개인환경설정(메모)

!!! question "메모 작성 시 본문 크기를 조절하는 방법이 있나요?"
    - 개인환경설정 화면에서 메모의 에디터 크기를 설정할 수 있음
    - 가로모드와 세로모드 중 선택이 가능하며, 사용자가 설정한 모드에 따라 메모 작성 및 조회 시 화면이 해당 크기로 표시됨

!!! question "메모 작성 시 자주 선택하는 수신자나, 과제카드 정보를 기본값으로 설정할 있나요?"
    - 개인수신그룹관리 화면에서 자주 사용하는 수신자를 그룹으로 등록해 두면, 메모 작성 시 수신자 선택 팝업의 개인 탭에서 해당 그룹을 선택하여 수신자를 한 번에 지정 가능
    - 개인환경설정 화면에서는 기본 수신자와 과제카드를 미리 설정할 수 있으며, 설정 후에는 별도의 선택 과정 없이 메모 작성 이 자동으로 지정된 상태로 시작됨
