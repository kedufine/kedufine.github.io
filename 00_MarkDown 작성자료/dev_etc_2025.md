# 2025년 문서관리 외 기능개선 내역

K-에듀파인 과제관리 등의 업무 기능개선 사항을 배포일자별로 세부사항을 제공합니다. 주요 기능 및 화면변경에 대해서만 작성하였고 오류수정 등은 포함하지 않습니다.

## 12월 23일

!!! success "[기록물이관] 기록물이관 파일생성 병렬처리 시도별 환경설정 기능 추가"
    - 기록관환경설정 화면의 이관배치시간설정 탭에서 업무시간 외 시간대로 이관배치 시간을 설정할 수 있도록 기능 추가
        1. 이관배치 시간이 업무시간 외로 설정된 경우만 **병렬처리 사용여부** 설정 값을 활성화 함
        2. **병렬처리 사용여부**가 사용으로 지정되면 병렬처리 세부설정(스레드 수, 청크 크기) 항목이 활성화 됨
        3. 사용자 설명 영역에 병렬처리 개념과 스레드 수, 청크 크기의 의미 및 설정 시 유의사항을 안내

??? success "[자료집계] 전체자료집계 화면에 서식함 기능 제공"
    - 2026년 1월 1일부로 집계서식생성 메뉴가 삭제됨에 따라 기존 메뉴에서 제공하던 서식함(서식함활용 팝업) 기능을 대체할 수 있도록 전체집계자료 화면에 서식함 기능 추가
    - 전체집계자료 화면에 추가된 서식함 기능은 과거처럼 등록된 서식을 확인, 활용할 수 있도록 **조회** 기능 제공

## 8월 26일

!!! success "[함께메모] 기능 신규 오픈"
    - 행정기관 업무관리시스템의 규격 변경에 따라 관련자 간 협업을 위한 **함께메모 기능이 추가**되었습니다.
    - **함께메모는 여러 사용자가 하나의 메모를 공동으로 작성하고 수정**할 수 있는 기능입니다.
    - 최초 작성자(보고자)가 등록한 메모는 설정된 기간 동안 참여자들이 본문과 첨부파일을 확인하거나 수정할 수 있습니다.
    - **동시 수정이 불가능**하므로, 다른 참여자가 수정 중인 경우 해당 사용자의 수정 저장이 완료된 이후에만 추가 수정이 가능합니다.
    - 메모 수정에는 **제한 시간이 설정**되어 있으며, **종료 1분 전 알림**이 표시됩니다. 제한 시간이 지나면 수정한 내용을 저장할 수 없으므로 반드시 제한 시간 내에 수정을 완료해야 합니다.
    - **8월 26일부터 부산 시범 운영 후 9월 전국으로 확대 예정**입니다.

??? success "[일정등록] 기본값 기관별로 설정하도록 개선"
    - **일정등록 시 기본값으로 설정되는 등록 시간을 기관별로 변경**할 수 있도록 개선되었습니다.
    - 기관업무계획담당자 권한을 가진 사용자가 **업무관리 > 일정관리 > 주월간업무계획 > 업무계획환경설정** 화면에서 일정시간설정을 변경하면, 해당 기관의 일정등록 시 변경된 시간이 기본값으로 적용됩니다.
    - 기존 설정값은 유지하면서, 시도교육청에서 선택할 수 있도록 옵션이 제공됩니다.

    - 주제영역 간 시각적 구분과 가독성 향상을 위해 레이아웃과 간격을 최적화하여 사용자 편의성 강화

    - 이를 해결하기 위해 과제카드함 화면에서 종료된 과제카드를 조회한 후, 담당자 변경 기능을 통해 참여자 정보를 수정할 수 있도록 개선
