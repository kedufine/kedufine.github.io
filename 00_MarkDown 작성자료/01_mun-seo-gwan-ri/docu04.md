# 개인정보필터

문서관리카드 상단의 `개인정보필터` 버튼을 이용하여 **본문과 첨부파일 안에 포함된 개인정보를 검출할 수 있습니다.** 또한 교육청 설정에 따라 결재 시 자동으로 개인정보 검출이 수행됩니다. 검출 결과는 안내 팝업을 통해 본문과 첨부파일별로 확인할 수 있습니다.


## 개인정보필터 검출 가능 항목

개인정보필터의 검출 항목은 교육청 총괄운영자가 설정하며, 모든 사용자는 이 설정을 따릅니다.

- 주민등록번호(필수)
- 카드번호
- 여권번호
- 운전면허번호
- 전화번호
- 핸드폰번호
- 이메일
- 보험번호
- 계좌번호
- 차량번호
- 성명
- 사업자명
- 법인번호
- 사업자번호


## 한건수정, 일괄수정 개인정보필터

문서함에서 문서를 선택한 후 한건수정 또는 일괄수정으로 **대국민공개여부를 비공개에서 공개로 전환**하는 경우에도 개인정보필터를 사용할 수 있습니다. 공개로 전환하려는 문서에 개인정보가 포함되어 있는지를 수정 팝업에서 바로 확인할 수 있습니다.

이 기능은 시도교육청의 문서환경설정 (1)에서 '한건수정, 일괄수정 개인정보필터 사용여부'를 설정한 경우에만 제공되므로, 소속 교육청의 설정에 따라 화면에 표시되지 않을 수 있습니다.
{ .annotate }

1.  :man_raising_hand: :material-sitemap-outline: 서비스공통 → 포털관리 → 운영환경관리 → 문서환경설정

!!! Info "문서관리카드 필터와의 차이"
    - 문서관리카드 : 대국민공개여부와 관계없이 본문과 첨부파일의 개인정보 검출
    - 한건수정, 일괄수정 팝업 : 선택한 문서 중 비공개에서 공개로 전환하는 문서만 개인정보 검출

!!! Info "검출 결과 확인 및 처리"
    - 검출상태 : 대기, 비대상, 검출, 미검출
    - 개인정보가 검출된 경우에도 대국민공개여부 수정 저장 가능
    - 검출 내역을 확인한 후 공개 여부를 판단하여 처리

한건수정, 일괄수정 기능을 제공하는 모든 문서함 (1)에서 동일하게 사용할 수 있습니다.
{ .annotate }

1.  :man_raising_hand: :material-sitemap-outline: 적용 화면
    - 업무관리 → 문서관리 → 문서함 → 문서등록대장
    - 업무관리 → 문서관리 → 문서함 → 부서인수문서
    - 업무관리 → 문서관리 → 내문서함 → 기안한문서
    - 업무관리 → 과제관리 → 나의과제
    - 업무관리 → 과제관리 → 부서과제
    - 업무관리 → 과제관리 → 과제카드관리 → 과제카드함
