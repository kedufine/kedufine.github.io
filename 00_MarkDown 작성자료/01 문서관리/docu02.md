# 마스킹

개인정보나 민감정보의 노출을 줄이기 위해 **문서관리카드 작성 시 `마스킹` 기능을 사용할 수 있습니다.** 문서관리카드 상단의 `마스킹` 버튼을 클릭하여 처리하며, 이 기능은 교육청 총괄운영자가 활성화한 경우에만 사용할 수 있습니다.

## 기능안내

- 마스킹 처리된 문서는 **문서를 열기 전 노출되는 영역에서는 마스킹된 제목이 표시**되며, 문서를 열면 원본이 출력됨
- **제목, 첨부파일명에 대해 설정할 수 있으며** 작성자가 마스킹하려는 단어에 직접 `*`을 입력해 변경
- 결재 처리 과정에서도 동일한 방법으로 마스킹 적용 가능
- 마스킹이 적용된 경우 문서를 열람하면 `"문서제목/첨부파일명에 마스킹이 설정되어 있습니다."` 안내 팝업이 표시됨
- 마스킹 시에는 **방향키, Shift, Space, Backspace, Del, `*`만 입력 가능**
- 결재 전 수행한 마스킹은 이후 **취소하거나 변경할 수 없음**, 결재 완료 후 수행한 마스킹은 처리과문서담당자가 마스킹 취소 가능
- 마스킹 처리된 문서는 **정보공개포털에는 마스킹된 상태로 공개**되고, 기록물 이관 시에는 원본 문서가 이관됨

!!! Info "결재완료된 문서 `마스킹` 처리 안내(후마스킹)"
	<figure markdown="span">

	- 문서등록대장에서 문서를 선택한 후 상단의 마스킹 버튼 클릭
	- 당해연도 문서에 한해 기안자만 처리 가능하며, 제목·첨부파일 제목·수기로 작성한 수신처에 대해 마스킹 가능
	- 마스킹 처리 내역은 DB에 기록되며 **처리과문서담당자(교육청 설정에 따라 결재선 포함)에게 내부메일로 발송**
	- **처리과문서담당자만 결재 완료 후 마스킹을 취소 가능**하며, 취소 시 최종 결재 버전의 제목으로 복원

## 제약사항

아래 조건에 하나라도 해당되는 문서의 경우 마스킹 기능이 적용되지 않습니다.

- 대국민공개여부 : 공개, 부분공개, 비공개(5호)	
- 직원열람범위 : 부서	
- 발송문서	
- 접수문서	
- 공동기안	
- 비전자문서(생산, 접수)
- 당해연도 이전 생산문서
