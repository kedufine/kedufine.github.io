# 문서관리카드 UI유형

개인문서환경설정에서는 문서관리카드 UI 유형을 3가지(`기본`/`가로`/`크게`) 형태로 변경하여 사용할 수 있습니다. K-에듀파인을 사용하는 다양한 사용자의 요구를 반영해 세 가지 형태의 문서관리카드 화면을 제공합니다.

:material-sitemap-outline: 서비스공통 → 개인설정 → 개인문서환경설정

## 기본

UI 유형 중 `기본`은 기존 문서관리카드 UI 형태를 그대로 유지한 것입니다. 사용자가 별도로 옵션을 변경하지 않았다면 이 형태로 문서관리카드가 표시됩니다. 이 UI 유형을 유지한 채 글자 크기만 변경할 수도 있으며, 글자 크기는 13, 16, 19 중 하나를 선택할 수 있습니다.

## 가로

UI 유형 중 `가로`는 결재정보와 본문을 한 화면에서 함께 볼 수 있는 옵션입니다. 가로형 모니터나 고해상도 모니터를 사용할 때 탭을 오가며 확인하는 방식보다 한 번에 모두 보여주는 형태가 더 효율적이어서 추가된 유형입니다.

  ![Image title](./images/img_02.png)
  <figcaption>UI유형 : 가로</figcaption>
</figure>

## 크게

마지막으로 UI 유형 중 `크게`는 결재 업무가 잦은 사용자를 위해 설계되었습니다. `가로` 형태와 마찬가지로 결재정보와 본문을 한 화면에서 볼 수 있으며, 상단 버튼 크기를 키우고 화면 좌측으로 배치해 조작성을 높였습니다. 또한 결재정보 영역은 중요한 내용을 스크롤 없이 최대한 한눈에 확인할 수 있도록 접기·펼치기 기능을 추가해 구성했습니다.
