#내부메일 자주묻는 질문

## 메일작성

:material-sitemap-outline: 내부메일 → 메일작성

!!! question "내부메일 주소를 잊어버렸는데, 다시 재설정하는 방법이 있나요?"
	내부메일 주소는 사용자가 직접 재설정할 수 없으며, **수신자 검색 기능(개인, 조직도, 사용자 그룹 조회)**을 통해 다시 확인해야 합니다.

!!! question "업무 메일을 발송했는데 수신자가 내용을 볼 수 없거나 빈 화면으로 보입니다 왜 그런가요?"
	해당 현상은 브라우저에 남은 캐시로 인해 메일 뷰어가 정상적으로 실행되지 않아 발생한 문제입니다.
    현재는 개선이 완료되어 해당 문제가 발생하지 않습니다.


!!! question "내부메일 작성 화면의 제목란이 회색으로 비활성화되어 메일을 작성할 수 없습니다. 어떻게 처리하나요?"
    해당 현상은 브라우저에 남은 캐시로 인해 메일 에디터가 정상적으로 실행되지 않아 발생한 문제입니다.
    현재는 개선이 완료되어 해당 문제가 발생하지 않습니다.


!!! question "내부메일 작성 시 '보안'이나 '예약 발신'과 같은 특별한 옵션들을 사용할 수 있나요?"
    내부메일 작성 시 **보안, 긴급, 발신메일저장, 수신자개별발송, 예약 발신** 등의 옵션을 선택할 수 있으며, 예약 발신은 배치 처리되므로 지정 시간보다 늦게 발송될 수 있습니다.


## 수신자 검색

:material-sitemap-outline: 내부메일 → 메일작성 → 수신자 검색

!!! question "메일 작성 시 타시도 사용자를 선택해서 발송할 수 있나요?"
    아니요. 내부메일은 외부 인터넷망과 연동되지 않고 **동일 시·도교육청 내부 사용자만 송수신 가능**하므로 타시도 사용자에게는 발송할 수 없습니다.


## 메일함 관리

:material-sitemap-outline: 내부메일 → 메일관리 → 메일함관리

!!! question "개인 메일함을 생성하고 관리하는 방법은 무엇인가요?"
    개인메일함은 메일함관리 화면의 개인메일함 영역에서 행추가 버튼을 클릭한 후, 메일함명을 입력하고 저장하면 생성됩니다.
    생성된 개인메일함은 메일 목록에서 선택하여 확인하거나, 필요 시 삭제 및 명칭 변경도 가능합니다.


## 메일서식관리

:material-sitemap-outline: 내부메일 → 메일관리 → 메일서식관리

!!! question "내부메일 서식을 등록, 수정, 삭제하여 사용할 수 있나요?"
    내부메일 메뉴의 메일서식관리 기능을 통해 서식을 등록·수정·삭제하여 사용할 수 있습니다

## 내부메일설정

:material-sitemap-outline: 내부메일 → 메일관리 → 내부메일설정

!!! question "수신하고 싶지 않은 내부메일이 계속 오는데, 특정 발신자를 차단할 수 있나요?"
    내부메일설정 화면에서 자동분류규칙을 설정하여 특정 발신자의 메일을 자동으로 분류할 수 있습니다.
    제목, 발신자, 본문 내용 등 조건에 맞게 분류규칙을 설정한 후, 해당 조건의 메일이 수신되면 이동시킬 메일함을 지정합니다.
    저장하면 설정한 규칙에 따라 해당 발신자의 메일이 자동으로 지정된 메일함으로 이동되어 사실상 차단됩니다.
    단, 시도별 운영 정책에 따라 자동분류규칙 기능이 제한될 수 있습니다.


!!! question "메일 작성시 하단에 자동으로 서명을 입력하고 싶습니다. 어디서 설정할 수 있나요?"
    내부메일 설정에서 **서명 자동포함 여부를 체크**하면, 메일 작성 시 하단에 지정한 서명이 자동으로 입력됩니다.


## 휴지통

:material-sitemap-outline: 내부메일 → 메일함 → 휴지통

!!! question "내부메일에서 삭제된 발신 메일 또는 휴지통의 메일을 복구할 수 있나요?"
    휴지통에 있는 메일은 ‘복구’ 버튼을 클릭하면 복구할 수 있으나, 휴지통 보관 기간이 지나 완전 삭제된 메일은 복구할 수 없습니다.

!!! question "내부메일에서 삭제된 발신 메일을 복구하고 싶어요. 어떻게 하나요?"
    삭제된 발신 메일은 **휴지통에 남아 있는 경우에만 복구 버튼으로 복구**할 수 있으며, 휴지통 보관 기간이 지나 **완전 삭제된 메일은 복구가 불가능**합니다.


## 차단메일함

:material-sitemap-outline: 내부메일 → 메일함 → 차단메일함

!!! question "차단한 메일 확인하고 싶습니다. 어디서 확인할 수 있나요?"
    차단한 메일은 내부메일 메뉴의 **차단메일함**에서 확인할 수 있으며, 제목을 클릭해 열람하거나 필요 시 복구·삭제할 수 있습니다.

## 수신메일함

:material-sitemap-outline: 내부메일 → 메일함 → 수신메일함

!!! question "사용자 계정이 삭제되면 개인 메일함이나 수발신 메일도 시스템에서 바로 삭제되나요? 재근무 시 메일을 다시 볼 수 있나요?"
    사용자 계정이 삭제되면 해당 계정의 **개인 메일함과 수·발신 메일은 시스템에서 함께 삭제**되며, 이후 재근무 시에도 **삭제된 메일은 복구하거나 다시 열람할 수 없습니다**.

## 내부메일 환경설정

:material-sitemap-outline: 서비스공통 → 포털관리 → 운영환경관리 → 내부메일 환경설정

!!! question "내부메일의 보관 기간은 어떻게 설정하며, 변경이 가능한가요?"
    내부메일의 보관 기간은 **시도총괄관리자가 내부메일 환경설정에서 발신·수신·휴지통·차단 메일함별 보관일수를 지정**하며, 필요 시 변경할 수 있습니다.

## 내부메일금칙어관리

:material-sitemap-outline: 서비스공통 → 포털관리 → 운영환경관리 → 내부메일 환경설정 → 내부메일금칙어관리

!!! question "메일 작성 시 특정 단어가 금칙어로 등록되어있어 메일 작성이 불가합니다. 금칙어는 무엇인가요?"
    금칙어는 **부적절한 단어나 홍보성 단어 등을 메일 본문·제목에서 사전에 차단하기 위해 등록된 단어**로, 내부메일 작성 시 입력되면 발송이 제한됩니다. 이는 **내부메일 환경설정의 금칙어 관리 기능**에서 시도총괄관리자가 설정합니다.


## 대용량송신사용자관리

:material-sitemap-outline: 서비스공통 → 포털관리 → 운영환경관리 → 내부메일 환경설정 → 대용량송신사용자관리

!!! question "학교장 직급의 교직원에게만 대량으로 내부 메일을 발송할 수 있나요?"
     내부메일 환경설정의 대량송신 사용자관리에서 학교장 직급의 교직원을 대상으로 대량송신 권한을 부여하면 해당 그룹에만 대량 메일을 발송할 수 있습니다
