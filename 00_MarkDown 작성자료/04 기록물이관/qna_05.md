#기록물이관 자주묻는 질문

## 기간설정

:material-sitemap-outline: 과제관리 → 기록물이관 → 기간설정

!!! question "2020년 기록물 이관 중에 2021년, 2022년 생산 기록물을 동시에 이관할 수 있나요?"
	2개 연도 동시 이관 불가 : 기록물 이관은 생산 연도별로 진행됩니다. 기간 설정 메뉴에서 이관할 생산 연도 선택 기능을 통해 특정 연도를 선택한 후 해당 연도의 기록물만 조회하고 이관을 진행하도록 되어 있습니다.

## 용량설정

:material-sitemap-outline: 과제관리 → 기록물이관 → 기록환경설정 → 용량설정

!!! question "이관파일사용현황의 용량은 실제 서버 용량과 다른가요? 이관 파일 용량이 부족한데, 어떻게 확보해야 하나요?"
	이관파일사용현황에 표시되는 용량은 실제 파일의 물리적 크기가 아닌, 데이터베이스(DB) 기준의 예상치로 계산된 값입니다.
    이관 요청을 진행하면 해당 요청의 용량이 DB상에서 누적되어 표시되며, 이는 파일 생성 전 미리 용량을 점검하기 위한 기능입니다.
    
	실제 서버의 사용 가능 용량은 기록관환경설정 화면의 용량설정 탭에서 확인할 수 있습니다.
    이관용량 또한 용량설정 탭에서 변경 가능합니다.
    이때 실제 서버의 여유 용량이 충분히 확보된 상태에서 수정이 이루어져야 합니다.

## 폐지조직관리

:material-sitemap-outline: 과제관리 → 기록물이관 → 기록환경설정 → 폐지조직관리

!!! question "폐지된 기관의 기록물은 어떻게 조치해야 하나요?"
	조직이 폐지되면 해당 조직에서 발생한 기록물은 이관 대행조직이 대신하여 관리해야 합니다.
	시도기록물담당자는 폐지된 조직의 기록물을 이관받을 대행 조직을 기록관환경설정 화면의 폐지조직관리 탭에서 설정합니다.
	
	이관 대행조직으로 지정된 조직은 이관처리카드수집, 기록물정리완료, 이관요청 등의 기록물 이관 업무를 대신 수행하게 됩니다.
	설정된 정보는 선택한 기관뿐만 아니라 소속된 하위 기관에도 동일하게 적용되며 유지됩니다.

## 비전자기록물소급일자등록

:material-sitemap-outline: 과제관리 → 기록물이관 → 기록환경설정 → 비전자기록물소급일자등록

!!! question "전년도 기록물 등록을 위한 비전자소급일자 수정은 어디서 하나요?"
	시도기록물담당자가 기록관환경설정 화면의 비전자기록물소급일자등록 탭에서 수정할 수 있습니다.
    문서등록대장에서 전년도 비전자 문서를 소급하여 등록할 수 있도록 하며, 시도에서 지정한 소급일자(**최대 5월 31일**)까지 등록 가능합니다.
	이 기간이 경과하면 비전자기록물의 소급 등록은 불가합니다.

## 미이관기록물정리함

:material-sitemap-outline: 과제관리 → 기록물이관 → 미이관기록물정리함

!!! question "기록물 추가이관은 언제 발생하는 것이고 미이관기록물정리함을 통해 어떻게 추가이관하는것인가요?"
	기록물 이관이 완료된 후, 누락된 기록물이나 미편철 과제카드 문서가 발생했을 때 기록물 추가이관을 진행됩니다.
    미이관기록물정리함 화면에서 미이관된 기록물을 처리할 처리과를 선택하고 이관과제카드수집을 진행합니다.
	추가 이관을 위한 최종 수집차수는 기존 차수에서 1차수가 더해져 반영되며, 최종 수집차수를 선택한 후 기록물정리완료 및 이관요청 순으로 추가이관을 진행합니다.

## 기록물정리함

:material-sitemap-outline: 과제관리 → 기록물이관 → 기록물정리함

!!! question "기록물정리현황에서 '미완료문서'를 클릭해도 목록이 조회되지 않거나, 미완료 상태가 계속 표시돼요."
	해당 문서 또는 과제카드를 이미 처리했음에도 불구하고, 다시 이관과제카드수집을 진행하지 않았기 때문입니다.
    문서 또는 과제카드 처리 완료 후, 다시 이관과제카드수집을 진행하면 정상적으로 처리 상태가 반영됩니다.
	만약 미완료 문서 목록 팝업창에서 기안일시, 접수일시 등 세부 정보가 표시된다면, 해당 문서의 내용을 확인하고 미완료 문서 화면에서 상태에 따라 문서를 처리한 다음 다시 한번 이관과제카드수집을 진행해야 합니다.

!!! question "학교에서 과제카드 이관업무 포함이 되나요?"
	- 학교는 기록물 이관 대상에 해당하며, 관련 업무를 수행합니다.
	- 기록물 이관은 K-에듀파인 시스템에서 생성된 기록물(문서, 메모)을 기록관으로 이관 처리하는 업무로, 공공 기록물 관리에 관한 법률에 근거하여 진행됩니다.
	- 학교의 기록물 이관 대상 포함 여부 : 기록물 이관 업무의 범위에는 기관 및 학교의 공공기록물 관리가 해당됩니다.
	- 부서/학교 기록물 담당자의 주요 업무 : 부서학교 기록물 담당자는 이관할 기록물을 확인하고 정리하여 이관을 요청하는 주요 역할을 담당합니다. 해당 담당자가 **기록물정리함** 메뉴에서 수행하는 주요 업무 단계는 다음과 같습니다.
		1. 문서 상태 체크: 이관을 진행해야 하는 미완료 문서가 존재하는지 확인합니다. (진행 중인 문서가 없어야 다음 단계 진행 가능)
		2. 이관 과제카드 수집: 이관 대상 단위 과제카드를 수집합니다. (과제카드 수집 상태는 **실행대기**로 변경되었다가 완료됨)
		3. 기록물 정리 완료: 기록물 정리를 완료했음을 설정하여, 추가적인 이관 과제카드 수집을 차단합니다.
		4. 이관 요청: 정리 완료된 기록물을 기록관으로 이관하도록 요청합니다. (기록물 정리 상태가 **정리완료**일 때만 이관 요청 가능)


## 기록물정리현황

:material-sitemap-outline: 과제관리 → 기록물이관 → 기록물정리현황

!!! question "폐지된 조직의 기록물이 정리현황에서 보이지 않아요. 어떻게 처리해야 하나요?"
	기본적으로 조회조건에 '미사용조직 검색제외' 체크박스가 선택되어 있기 때문입니다.
	이 경우 미사용 조직에 대한 정리 상태는 조회되지 않습니다.
    '미사용조직 검색제외' 체크박스를 해제하면, 미사용 조직의 기록물 정리 상태도 조회할 수 있습니다

## 이관실행현황

:material-sitemap-outline: 과제관리 → 기록물이관 → 이관실행현황

!!! question "기록물 이관 시 '실행대기' 상태에서 다음 단계로 넘어가지 않고 계속 멈춰요. 어떻게 처리해야 하나요?"
	기간 설정 또는 이관 연도가 상이하여 발생합니다. 아래 내용에 따라 처리하시면 됩니다.
    
	- 이관 대상 부서의 기간 설정 문제 : 과제관리 > 기록물이관 > 기간설정 화면에서 이관하려는 날짜가 포함될 수 있도록 기간을 변경해야 합니다.
	변경 후에는 파일 생성 배치가 자동으로 처리됩니다.
	이관 기간은 **시작 일자와 종료 일자** 사이에서만 파일 생성이 가능하며, 기간 설정이 없거나 누락된 경우 파일 생성이 되지 않거나 실행대기 상태로 남을 수 있으므로 반드시 사전에 설정해야 합니다.
    
	- 과거 연도 이관 요청 또는 승인처리 문제 : 과거 연도에 이관 요청 및 이관 승인 처리를 미리 진행하여 이관 연도가 현재 연도가 아닌 경우에는 **이관초기화**를 진행한 후 이관을 다시 진행해야 합니다.
	이관 초기화는 과제관리 > 기록물이관 > 이관실행현황 화면에서 할 수 있습니다.

!!! question "RMS에서는 인수 완료인데 K-에듀파인에서는 여전히 '접수' 또는 '파일생성' 상태인 경우 어떻게 해야 하나요?"
	일반적으로 K-에듀파인 시스템이 RMS(기록관리시스템)로부터 ACK(인수확인 응답)를 정상적으로 수신하지 못할 경우 발생합니다.
	이관실행현황 화면에서 직접 인수완료 처리하시면 됩니다.

!!! question "전자기록물 이관 전에 시스템 오류 방지를 위해 일괄 보정을 요청하고 싶은데, 언제 해야 하나요?"
    전자기록물 이관 전에는 시스템 오류를 막기 위해 **최소 2~3주 전**에 사전 환경설정 확인 및 보정을 요청해야 합니다

!!! question "기록물 이관 중 오류가 발생하여 다시 처음부터 진행하려면 어떻게 해야 하나요?"
    기록물 이관 중 오류가 발생했을 경우, 이관실행현황 화면에서 **이관초기화**를 진행한 뒤 기록물정리현황에서 ‘기록물정리완료취소 > 과제카드수집 > 기록물정리완료 > 이관요청’을 다시 수행해야 처음부터 재이관할 수 있습니다.

!!! question "기록물 이관 진행 중 '실패' 상태가 발생했는데, 어떻게 해결해야 하나요?"
    기록물 이관 중 ‘실패’ 상태는 파일 생성 과정에서 오류가 발생한 경우이며, 이때는 이관초기화 후 다시 과제카드(문서) 수집부터 순차적으로 진행해야 합니다.

!!! question "K-에듀파인에서 이관 완료된 기록물의 수량이 RMS와 일치하지 않는 경우 어떻게 해야 하나요?"
    이관 완료 후 K-에듀파인과 RMS(기록관리시스템)의 기록물 수량이 불일치하면, 기록물이관수량검증 메뉴에서 처리과별·과제카드별 비교를 진행하고, 차이가 확인되면 해당 자료를 첨부해 사용자지원서비스에 등록해야 합니다.

## 처리과별실적건수

:material-sitemap-outline: 과제관리 → 기록물이관 → 처리과별실적건수

!!! question "처리과별실적건수 메뉴의 문서 수와 이관실행현황 메뉴의 문서 수가 다른데, 왜 그런가요?"
    처리과별실적건수는 이관 전에 수집된 기록물 건수를 보여주고, 이관실행현황은 실제 파일 생성 및 이관 처리된 건수를 기준으로 하므로 시점 차이나 정리 과정에서 불일치가 발생할 수 있습니다.
