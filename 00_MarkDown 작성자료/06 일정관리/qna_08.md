#일정관리 자주묻는 질문

## 일정현황

:material-sitemap-outline: 업무관리 → 일정관리 → 일정현황

!!! question "등록된 일정을 달력이나 목록 형태로 조회하려면 어떻게 해야 하나요?"
	- 등록된 일정은 일정현황 메뉴에서 캘린더(월/주/일) 화면이나 목록 조회 화면을 통해 확인할 수 있으며, 제목을 클릭하면 상세일정도 조회할 수 있습니다.

!!! question "기관장 일정을 등록하고 하위 조직과 공유하려면 어떻게 해야 하나요?"
	- 기관장 일정은 교육청일정담당자만 등록 가능하며, 등록 후에는 공개 범위를 기관 전체나 하위 조직까지 설정하여 공유할 수 있습니다.

## 일정작성 일괄파일업로드

:material-sitemap-outline: 업무관리 → 일정관리 → 일정현황 → 일정작성 일괄파일업로드

!!! question "여러 일정을 한 번에 일괄 등록하는 방법이 궁금합니다."
    - 일정현황 화면에서 파일업로드 기능을 사용합니다.
    일괄작성 팝업창이 열리면 서식을 다운로드하여 일정 내용을 입력한 뒤 저장합니다.
    작성이 완료된 파일을 다시 업로드하고 내용을 확인한 후 저장하면, 일괄 등록이 완료됩니다.

## 기관업무계획

:material-sitemap-outline: 업무관리 → 일정관리 → 주월간업무계획 → 기관업무계획

!!! question "기관 업무계획을 관리하고 제출하는 절차는 어떻게 되나요?"
    - 기관업무계획은 **기관업무계획담당자가 부서 업무계획을 확인 후, 기관 단위 계획을 수립·작성하여 저장**하는 절차입니다. 이때 업무계획 정보 입력 후 파일을 첨부해 제출하며, 기관 업무계획 서식 파일은 1개만 등록할 수 있습니다.

## 나의과제일정

:material-sitemap-outline: 업무관리 → 일정관리 → 과제일정 → 나의과제일정

!!! question "일정에 연동하여 문서나 메모를 작성할 수 있나요?"
    - 일정관리에서 등록된 일정과 연동하여 문서, 메모, 일지 등의 실적을 작성하고 관리할 수 있습니다.
    나의과제일정 화면에서 문서 또는 메모와 연동할 일정을 선택한 후, 문서작성 또는 메모작성 버튼을 클릭합니다.
    업무에 맞게 기안을 완료하면 해당 일정과 문서(또는 메모)가 연동됩니다.
    단, 문서작성은 과제카드가 연결된 일정에서만 가능합니다.
