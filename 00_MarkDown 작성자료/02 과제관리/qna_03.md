#업무인계인수 자주묻는 질문

## 인계함

:material-sitemap-outline: 업무인계인수 → 인계함

!!! question "업무 인계인수 시 인계자의 영구열람제한 문서는 인수자에게 어떻게 공유되나요?"
	업무 인계인수 시 인계자가 작성한 영구열람제한 문서는 인수자에게 자동으로 보안 처리되어 공유됩니다.
	인계자가 해당 문서를 인계하고 싶지 않다면 인계신청시 업무현황에서 제외할 수 있습니다.
	인수자는 인계자가 제외한 목록은 확인할 수 있지만, 해당 문서의 상세 내용은 열람할 수 없습니다.

!!! question "업무 인계인수 시 인계자가 선택한 과제카드의 인수자는 자동으로 과제담당자로 등록되나요?"
	업무 인계인수 시, 인계자는 과제카드에 등록된 본인의 담당 권한을 인수자에게 이관합니다.
	인계자가 해당 과제카드의 과제담당자 권한을 가진 상태로 아래 절차가 완료되면, 인수자는 과제카드의 **과제담당자**로 자동 지정됩니다.
	
	1. 입회자의 입회확인 완료
	2. 인수자가 인계 부서로 인사이동
	3. 익일 새벽 시스템 배치 동작(인수자·인계자 정보 확인) 시점에 자동 적용

!!! question "업무인계인수 시 과제관리자가 후임에게 인계되지 않는 문제와 관련하여 과제카드 담당자/관리자 권한은 어떻게 처리되나요?"
	업무인계인수 시, 과제카드 담당자 및 관리자 권한은 인계 대상에 포함됩니다.
	인수자가 인계자의 부서로 이동하고, 입회자의 확인이 완료되어야 최종 완료 상태가 되며, 익일 새벽 시스템 배치에 의해 업무 인계인수 정보가 반영되어 과제카드 담당자/관리자 권한이 자동 적용됩니다.	
	만약, 인계자가 동일 부서 내에 계속 존재하는 경우 아래와 같이 권한 변경이 발생합니다.
	인계자가 과제카드 관리자 권한을 보유하고 있다면 인수자에게 해당권한이 이양되고, 인계자는 과제카드 담당자로 변경됩니다.
	인계자가 과제카드 담당자 권한을 보유하고 있다면 인수자, 인계자 모두 과제카드 담당자로 지정됩니다.
	다만, 인계자가 부서이동한 경우 인수자만 권한을 가지게 됩니다.		
	과제카드별 담당자 또는 관리자 권한을 수동으로 변경하려면, 과제관리 > 과제카드관리 > 과제카드함 메뉴에서 해당 과제카드를 선택한 후, 담당자변경 버튼을 통해 직접 수정할 수 있습니다.

!!! question "업무인계인수 시 파견 중인 직원을 인수자로 선택할 수 없나요?"
	업무인계인수 시 파견 중인 직원을 인수자로 지정할 수 있습니다.
	다만, 조직도에 파견 기관이 조회되는 경우에만 인수자로 조회·지정 가능하며, 조직도에 없을 경우 인수자로 지정할 수 없습니다.

!!! question "신규 발령된 직원을 업무인계인수의 인수자로 선택할 때 조회되지 않는 이유는?"
	현재 소속 조직(부서)에 정상적으로 등록된 사용자만 인수자로 선택 가능합니다.
	해당 직원의 현재 소속을 확인한 후 인수 진행하시기 바랍니다. 


## 입회함

:material-sitemap-outline: 업무인계인수 → 입회함

!!! question "업무인계인수 최종 완료 후에도 인계자에게 단위과제카드가 그대로 남아있고 과제담당자로 지정되어 있는 문제가 발생하는데 해결방법은?"
	인계자가 동일 부서에 계속 남아 있는 경우 과제카드 담당자 권한이 유지되며, 인수자에게 동일한 권한이 부여됩니다.
	인계자의 권한은 인사이동시 제거됩니다.		
	만약, 인계자가 인사이동 전 권한을 제거해야하는 직접 수정하셔야합니다.
	과제카드함에서 대상 카드를 열어 표제부 탭의 과제카드 담당자 영역의 변경 버튼으로 담당자를 수정합니다.
	과제카드함 목록에서 여러 카드를 선택해 담당자일괄처리로 담당자를 한 번에 지정할 수도 있습니다.

!!! question "인수자가 발령일자에 맞춰 부서이동을 했음에도 업무인계인수 최종 완료가 되지 않고 과제담당자로 변경되지 않는 문제가 발생하는데 이유는?"
	인계자, 인수자, 입회자 중 누구 하나라도 **확인**을 누르지 않으면, 인계서가 진행 중 상태로 남아 업무인계인수가 처리되지 않습니다.
	또한, 입회자의 입회 확인후 익일 새벽 1시 인수인계가 자동 처리되며, 이때 부서 이동상태를 확인하므로 입회 확인과 인수자의 부서를 확인하시기 바랍니다. 
	

## 부서(학교)업무인계인수함

:material-sitemap-outline: 업무인계인수 → 부서(학교)업무인계인수함

!!! question "업무인계인수 건의 입회자를 변경하고 싶습니다."
	인수자, 인계자별로 인수확인취소, 인수요청취소 등의 기능을 이용하여 인수요청상태를 **작성중**으로 되돌린 후 인계신청 상세에서 입회자를 변경합니다.		
	권한자의 경우 업무인계인수 > 업무인계인수관리 > 부서업무인계인수관리함 화면에서 담당업무를 선택한 후 인계신청상세조회 팝업에서 입회자 변경하고 저장하시면 됩니다.
	진행 상태가 인수요청 또는 인수확인인 인계인수만 변경 가능합니다.
