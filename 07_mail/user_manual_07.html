    내부메일 사용자교육자료

    내부메일 사용자교육자료

    개요

    내부메일은 K-에듀파인 시스템 내에서 효율성을 높이기 위해 사용자 간 메일 송수신 기능입니다. K-에듀파인 시스템은 폐쇄망으로 운영되므로 외부 인터넷 메일과는 연동되지 않습니다.

    메일작성

    메일작성 화면은 사용자가 특정 대상에게 내부메일을 작성하여 발송하는 공간입니다. 이 화면은 제목, 수신자, 발송 옵션 등을 설정하는 메일 정보 영역과 내용을 입력하는 본문 영역으로 구성되어 있습니다.

    본문 작성 및 발송

    사용자는 메일의 제목과 본문을 입력하고 필요한 경우 파일을 첨부하여 메일을 보낼 수 있습니다. 메일 작성을 완료한 후에는 발신 버튼을 클릭하여 즉시 메일을 발송하거나 임시저장 버튼을 클릭하여 나중에 이어서 작성할 수 있습니다.

    메일 발송 시 제약사항

    • 시도총괄관리자가 설정한 금칙어가 제목이나 본문에 포함된 경우 메일 발송이 제한
    • 메일 본문의 용량이 시스템에 설정된 최대 용량을 초과하면 메일을 발송되지 않음
    • 예약 발신 옵션 선택 후 발송 시 순차 배치 처리 방식 때문에 예약 시간보다 발송이 다소 늦어질 수 있음음

    메일 작성 주요기능 안내

    • 미리 등록해 둔 서명을 메일 본문 하단에 자동으로 입력 가능, 서명은 최대 4개까지 등록할 수 있으며 그중 하나를 기본 서명으로 지정 가능
    • 발송 전 작성한 메일이 수신자에게 어떻게 보일지 미리 확인 가능
    • 작성 중인 메일을 임시보관함에 저장하여 추후에 수정하거나 발송할 수 있음

    메일 발송 옵션 주요기능 안내

    • 보안 옵션을 선택하면 수신자가 인증서로 로그인해야만 메일 내용을 확인 가능, 보안메일에 답장하거나 전달할 경우 보안 옵션은 해제
    • 긴급 옵션을 적용한 메일은 수신자의 메일함에서 수신 일시와 관계없이 목록 최상단에 표시
    • 발신메일 저장은 발송한 메일을 자신의 발신메일함에 저장하는 기능, 메일 회수 등의 기능을 사용하기 위해 선택하는 것을 권장
    • 수신자 개별 발송 옵션 선택 후 여러 명에게 메일을 보낼 때 각 개인에게 개별적으로 발송
    • 예약 발신 옵션은 지정한 날짜와 시간에 메일이 예약발송되도록 설정

    수신자 지정

    메일을 받을 대상을 받는사람, 참조, 숨은참조로 구분하여 지정할 수 있습니다. 수신자 지정은 아래와 같은은 방법을 통해 가능합니다.

    • 사용자 검색, 개인 수신 그룹, 조직도 등의 탭을 활용하여 수신자를 찾아 추가
    • 엑셀 서식을 다운로드하여 수신자 정보를 입력한 후 파일을 업로드하는 방식으로 다수의 수신자를 한 번에 일괄로 추가

    임시보관함

    임시보관함은 사용자가 작성 중에 임시저장한 메일의 목록을 확인하고 관리하는 공간입니다. 이곳에서 메일 작성을 이어서 하거나, 더이상 불필요하면 메일을 삭제할 수 있습니다.

    임시보관된 메일의 제목을 클릭하면 메일작성 화면으로 이동하여 내용을 이어서 작성할 수 있습니다. 재작성 화면에서는 메일을 발송하거나 다시 임시저장하는 것이 가능합니다.

    임시보관 메일 목록 관리방법 안내

    • 목록에서 선택한 메일을 메일다운로드 또는 파일 버튼을 눌러 사용자의 PC로 저장 가능
    • 조회된 임시보관 메일 목록을 출력 버튼을 눌러 인쇄 가능능
    • 선택한 메일을 삭제 버튼을 눌러 휴지통으로 이동할 수 있음
    • 선택한 메일을 완전삭제 버튼을 눌러 휴지통을 거치지 않고 즉시 영구적 삭제할 수 있음

    메일함

    미확인수신메일함

    미확인수신메일함은 사용자가 수신한 메일 중 아직 확인하지 않은 메일의 목록을 확인하는 공간입니다. 이 화면에서 메일을 직접 확인하지 않고도 읽음 처리하거나 중요 메일을 별도로 보관하는 등 목록을 관리할 수 있습니다.

    수신메일함

    수신메일함은 사용자가 수신한 모든 메일의 목록을 확인하고 관리하는 공간입니다. 이 화면에서는 메일 목록을 관리하고 메일 내용을 상세히 조회하여 답장, 전달 등의 후속 작업을 수행할 수 있습니다.

    발신메일함

    발신메일함은 사용자가 발송한 메일의 목록을 확인하고 관리하는 공간입니다. 이 화면은 메일 작성 시 발신메일저장 옵션을 선택한 메일만 조회됩니다.

    개인메일함

    개인메일함은 사용자가 수신메일함이나 발신메일함에서 중요한 메일을 별도로 보관하기 위해 이동시킨 메일 목록을 확인하고 관리하는 공간입니다. 사용자는 '메일함관리' 메뉴를 통해 개인메일함을 직접 생성할 수 있으며, 다른 메일함과 달리 개인메일함으로 옮겨진 메일은 설정된 보관 기한이 지나도 자동으로 삭제되지 않는 특징이 있습니다.

    차단메일함

    차단메일함은 사용자가 내부메일설정에서 등록한 자동분류규칙(자동필터)에 따라 특정 조건의 메일이 자동으로 이동되어 보관되는 공간입니다.

    휴지통

    휴지통은 각 메일함에서 사용자가 삭제한 메일이 영구 삭제되기 전 임시로 보관되는 공간입니다. 사용자는 삭제했던 메일을 최종적으로 처리하기 전에 이곳에서 목록을 확인하고 복구하거나 영구 삭제할 수 있습니다.

    메일함 기능

    작성 중

    메일관리

    메일함관리

    메일함관리는 내부메일 사용량을 확인하고, 중요한 메일을 영구 보관하기 위한 개인메일함을 관리하는 공간입니다. 사용자는 이 화면에서 메일함의 상태를 점검하고 필요에 따라 개인메일함을 추가하거나 기존 메일함의 데이터를 일괄 처리할 수 있습니다.

    메일함관리 화면은 내부메일 사용량, 기본메일함, 개인메일함 3개 영역으로 구분됩니다.

    • 내부메일 사용량

      사용자는 자신의 전체 메일 용량과 현재 사용량을 확인할 수 있습니다. 만약 설정된 용량을 초과하면 메일 수신 및 발신이 제한될 수 있습니다.

    • 기본메일함

      기본메일함(수신메일함, 발신메일함 등) 목록 및 정보(매일개수, 사용량(MB))를 조회할 수 있습니다. 비우기 버튼을 통해 선택한 메일함의 모든 메일을 한 번에 휴지통으로 이동시킬 수 있습니다. 휴지통을 선택하고 '비우기'를 실행하면 해당 메일들은 영구 삭제됩니다.

    • 개인메일함

      개인메일함 목록 및 정보(분류규칙설정여부 (1) , 매일개수, 사용량(MB))를 조회할 수 있습니다. 행추가, 행삭제 (2) 버튼을 통해 개인메일함을 직접 생성, 수정, 삭제할 수 있습니다.

      1. 자동분류규칙에 의해 사용되고 있는지 여부
      2. 기본 메일함은 삭제 불가능

    내부메일설정

    내부메일설정은 개인의 메일 사용 환경을 직접 설정하는 공간입니다. 시도총괄관리자가 설정한 범위 내에서 사용자는 자신의 편의에 맞게 설정을 변경할 수 있습니다. 내부메일설정, 자동분류규칙설정으로 구분됩니다.

    • 내부메일설정

      발신메일, 수신메일함, 휴지통, 차단메일함의 보관일수를 지정할 수 있고 최대보관일수는 15일 입니다. 보관일수는 시도교육청에서 설정한 최대 기간을 넘을 수 없으며, 설정된 기간이 지나면 메일은 휴지통으로 자동 이동되거나 완전 삭제됩니다.

      서명 관리 주요 기능

      • 메일 작성 시 본문 하단에 자동으로 추가될 서명을 관리할 수 있음음
      • 최대 4개까지 서명을 등록할 수 있습니다.

      • 등록된 서명 중 하나를 기본 서명으로 지정하면 메일 작성 시 해당 서명이 자동으로 입력 (1)

        1. 서명자동포함여부를 체크해야 자동 서명 기능이 동작

    • 자동분류규칙 관리

      특정 조건에 맞는 메일이 수신될 때 지정된 메일함(차단메일함 또는 개인메일함)으로 자동으로 이동되도록 규칙을 설정하는 기능입니다.

      자동분류규칙 설정 방법

      1. 규칙 추가: 행추가 버튼을 클릭한 후, 분류 기준으로 제목, 발신자, 본문 내용 중 하나를 선택하고 이동시킬 메일함을 지정하여 저장
      2. 규칙 삭제: 설정된 자동분류규칙을 선택하여 삭제
      3. 발신자 기준 설정 제약: 발신자를 기준으로 규칙을 설정하는 기능은 시도총괄관리자가 해당 옵션을 사용으로 설정한 경우에만 활성화

    메일서식관리

    메일서식관리는 사용자가 자주 사용하는 메일 양식을 서식으로 만들어 관리하는 공간입니다. 이를 통해 반복적인 메일 작성 업무를 간소화할 수 있습니다.

    권한 종류

    일반사용자

    내부메일 시스템의 기본적인 기능을 사용하는 사용자로, 자신의 메일 송수신 및 개인 환경설정을 관리합니다.

    • 메일 작성, 발송, 수신 및 각종 메일함 조회
    • 보안, 긴급, 예약 발신 등 발송 옵션 사용
    • 시도총괄관리자가 설정한 범위 내에서 개인의 메일함 보관일수, 자동 분류 규칙, 서명 등 개인 환경 설정
    • 중요 메일 보관을 위한 개인메일함 생성 및 관리
    시도총괄관리자

    해당 시도 교육청의 내부메일 시스템 전반에 대한 운영 정책을 설정하고 관리하는 권한을 가집니다.

    • 시스템 운영 설정: 사용자별 기본 메일 용량, 최대 첨부파일 크기, 메일함별 최대 보관일수, 최대 수신자 수 등 시스템의 기본 운영 환경 설정
    • 발신 제한 사용자 관리: 특정 사용자의 메일 발송을 제한하거나 해지
    • 금칙어 관리: 메일의 제목이나 내용에 포함될 경우 발송을 차단할 금칙어를 등록하고 관리
    • 대량송신 사용자 관리: 기본 최대 수신자 수를 초과하여 메일을 발송할 수 있는 특별 사용자를 지정하고 관리
    대량송신 사용자

    시도총괄관리자에 의해 지정되어, 시스템에 설정된 최대 수신자 수를 초과하여 메일을 발송할 수 있는 권한을 부여받은 일반 사용자입니다.
