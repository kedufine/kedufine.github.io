    2025년 기능개선 내역

    K-에듀파인 업무관리 업무영역의 기능개선 사항을 배포일자별로 세부사항을 제공합니다. 주요 기능 및 화면변경에 대해서만 작성하였고 오류수정 등은 포함하지 않습니다.

    11월 25일

    [문서등록대장] 한건수정과 기록물이관 워터마크 표기 관련 개선
    • 문서관리카드 한건수정 시 대국민공개여부 수정시에만 대국민공개여부 수정사유를 입력하도록 개선
    • 기록물이관 완료된 문서 조회 시 문서관리카드 ui유형(크게, 가로)도 워터마크가 표기되도록 개선
    [문서등록대장] 감사열람조직 조회 시 문서와 첨부물분리등록의 열람권한이 일치되게 개선
    • 감사담당자관리에서 등록한 감사담당자 권한으로 감사열림조직 조회 시 첨부물분리등록 첨부파일도 열람되도록 개선
    • 감사열람조직 사용 시 일치된 열람권한을 제공하여 원활한 업무처리 지원
    [문서진행] 문서진행 목록의 기안(담당)자 항목 클릭 시 정렬이 되도록 개선
    • '문서진행' 목록의 기안(담당)자 기준으로 오름차순, 내림차순 정렬기능 추가

    10월 28일

    [미완료문서] 미접수 문서 조회 및 일괄접수 기능 추가
    • '미완료문서' 화면에서 조회유형을 미접수로 선택 후 조회 시 미접수 상태의 문서를 선택하여 일괄접수할 수 있는 기능을 추가
    • 일괄접수 기능 추가를 통해 '접수대기' 화면으로 이동하지 않고 바로 접수가 가능하여 업무 편의성 향상
    [통합검색] 휴직자 권한 보유자가 통합 검색 기능 사용 시 대국민공개여부 공개 문서만 통합 검색이 가능하도록 기능개선
    • 휴직자 권한 보유자가 통합검색 기능 이용 시 대국민공개여부가 공개인 문서를 조회할 수 있도록 기능 개선
    • 기존에는 휴직자가 문서를 확인할 수 없었으나, 공개 문서에 한해 통합검색을 통한 조회가 가능해져 문서 확인 편의성 향상
    [문서등록대장(다년도검색)] 키워드 검색 안내 팝업 추가
    • '문서등록대장(다년도검색)' 화면에서 검색 시 사용 가능한 키워드 항목을 안내하는 팝업을 추가하여 사용자가 검색 조건을 쉽게 파악하고 활용할 수 있도록 개선
    • 제목 조건 오른쪽에 검색도움말 버튼을 추가하여 키워드 검색 방법을 안내하고 검색 편의성을 향상
      • 키워드 항목 : AND 검색, OR 검색, 복합 검색, NOT 검색, 순서 검색 1, 순서 검색 2

    9월 23일

    [개인수신그룹관리] 대내수신자설정여부가 아니요 인 기관을 개인수신그룹으로 등록 시 대내조직 등록 금지
    • 수신그룹관리, 개인수신그룹관리 화면에서 엑셀 파일을 통한 개인수신그룹 일괄등록 시 대내수신자설정여부가 아니오로 설정된 조직은 미등록
    • 수신그룹관리, 개인수신그룹관리에 엑셀을 업로드할 경우 대내수신자설정여부 확인 후 등록되도록 개선하여 발송처리 시 수신실패가 발생하지 않도록 변경
    [문서등록대장] 문서관리카드 내 수정 기능 추가
    • 문서등록대장 화면에서 문서관리카드 클릭 시 상단 수정 버튼을 추가하여 문서 내용을 확인한 후 바로 수정할 수 있도록 개선
    • 기존 문서등록대장 목록에서 제공하던 한건수정과 동일 기능
      • 수정 가능항목 : 과제카드명, 관련정보, 대국민공개여부, 직원열람범위, 직원열람제한, 쪽수
    [문서진행] 문서 재작성 처리 시 결재선에 공석이 존재할 경우 안내문구 추가
    • 문서진행 화면에서 문서 체크 후 상단 재작성 클릭 시 결재경로설정 팝업과 함께 공석관련 안내문구를 추가하여 공석자가 포함된 결재선을 변경하여 처리할 수 있도록 안내
    • 생상문서는 반려, 회수만 재작성 가능
    • 안내문구 : 결재경로에 공석(공석사유)이 포함되어 있습니다. 사용자의 공석정보를 확인 후 결재경로를 변경하여 처리하여 주시기 바랍니다.
    [공용서식] 문서관리카드 내 공람 인원수 표기 추가
    • 공용서식, 결재대기, 문서진행, 문서등록대장 등의 화면에서 문서관리카드 조회 시 공람탭에 인원수를 표기하여 공람여부를 바로 알 수 있도록 개선
    • 문서관리카드를 확인 할 수 있는 모든 문서함 적용
    [문서등록대장] 문서관리카드 내 문서번호복사 기능 사용 시 날짜 표기형식 개선
    • '문서등록대장' 화면의 문서관리카드 내 문서번호복사 기능 사용 시 날짜 표기형식을 개편된 행정업무편람에 맞추어 있도록 개선
    • 표기형식 : 2025. 9. 9 (O) / 2025. 09. 09 (X)
    [공용서식] 공문게시지정 시 주제영역의 전체 선택항목 삭제
    • 공용서식, 결재대기, 접수대기 등의 화면에서 문서처리 시 공문게시지정 기능 사용 중 주제 영역의 전체 선택항목 삭제

    8월 26일

    [발송처리] 수신자숨김의 경우 발송처리에서 수신자 항목이 마스킹 되도록 개선
    • '발송처리' 화면에서 마스킹된 항목을 볼 수 있는 사용자라도 수신자 항목은 항상 마스킹되도록 개선
    [재정기안] 재정문서 중 학교(지출품의)의 경우 대국민공개여부의 초기값을 설정하지 않아 사용자가 선택할 수 있도록 개선
    • 재정문서의 대국민공개여부 초기값은 '비공개'로 설정되어 있었지만 학교(지출품의)에 한해 초기값을 지정하지 않고 사용자가 직접 선택할 수 있도록 개선

    [발송대기] 문서함보유여부와 대내수신자설정여부 '아니오' 설정 시 문서 발송처리되지 않고 수신실패 되도록 개선

    • '발송대기' 화면에서 문서 발송처리 시 수신처 중 문서함보유여부와 대내수신자설정여부에 따라 수신실패 되도록 개선
    • 수신실패 조건 : 문서함보유여부와 대내수신자설정여부 중 하나라도 '아니오'면 수신실패
    • 국립학교 등 문서함보유여부와 대내수신자설정여부가 정상적으로 설정되어 있는지 점검 필요
    [배부대기] 문서 배부처리 시 조직트리에 문서함보유여부와 대내수신자설정여부를 확인하여 생성하도록 개선
    • '배부대기' 화면에서 문서 배부처리 시 배부처 지정 과정에서 문서함보유여부와 대내수신자설정여부 설정에 따라 배부처 조회 가능
    • 배부처에 조회되는 조건 : 문서함보유여부와 대내수신자설정여부 중 하나라도 '아니오'면 조회 안됨
      • 예외 조건 : 문서함보유여부와 대내수신자설정여부가 '예'로 설정된 하위부서가 있는 경우 조회 됨
    [문서진행] 국민신문고 문서를 수정 후 기안 시 회수하면 초기 문서가 아닌 수정된 문서로 반영되게 개선
    • '문서진행' 화면에서 국민신문고 결재문서를 수정 후 회수할 경우, 수정내용이 정상적으로 반영되도록 개선

    [함께메모] 기능 신규 오픈

    • 행정기관 업무관리시스템의 규격 변경에 따라 관련자 간 협업을 위한 함께메모 기능이 추가되었습니다.
    • 함께메모는 여러 사용자가 하나의 메모를 공동으로 작성하고 수정할 수 있는 기능입니다.
    • 최초 작성자(보고자)가 등록한 메모는 설정된 기간 동안 참여자들이 본문과 첨부파일을 확인하거나 수정할 수 있습니다.
    • 동시 수정이 불가능하므로, 다른 참여자가 수정 중인 경우 해당 사용자의 수정 저장이 완료된 이후에만 추가 수정이 가능합니다.
    • 메모 수정에는 제한 시간이 설정되어 있으며, 종료 1분 전 알림이 표시됩니다. 제한 시간이 지나면 수정한 내용을 저장할 수 없으므로 반드시 제한 시간 내에 수정을 완료해야 합니다.
    • 8월 26일부터 부산 시범 운영 후 9월 전국으로 확대 예정입니다.
    [일정등록] 기본값 기관별로 설정하도록 개선
    • 일정등록 시 기본값으로 설정되는 등록 시간을 기관별로 변경할 수 있도록 개선되었습니다.
    • 기관업무계획담당자 권한을 가진 사용자가 업무관리 > 일정관리 > 주월간업무계획 > 업무계획환경설정 화면에서 일정시간설정을 변경하면, 해당 기관의 일정등록 시 변경된 시간이 기본값으로 적용됩니다.
    • 기존 설정값은 유지하면서, 시도교육청에서 선택할 수 있도록 옵션이 제공됩니다.

    [감사담당자열람이력] 조회 목록과 엑셀다운로드의 결과 값이 일치하도록 개선

    [접수대기] 일괄접수 팝업에서 접수버튼 하단에 나의결재선 기능이 위치하도록 개선
    • '접수대기' 화면의 일괄접수, 일괄담당지정 팝업 상단 신규 추가된 나의결재선 기능의 위치를 접수버튼 하단으로 변경하여 기존 접수버튼의 위치가 익숙한 사용자들의 혼란을 최소화 하도록 개선
    • 일괄접수 팝업 : 접수, 결재경로 버튼 하단에 나의결재선 위치
    • 일괄담당지정 팝업 : 결재경로 버튼 하단에 나의결재선 위치
    대용량별송파일 다운로드 기능 추가

    [문서환경설정] 문서환경설정에 개인정보필터 사용여부에 사용안함 설정 추가 개선
    • '문서환경설정' 화면에 사용안함 항목을 추가하여 개인정보필터 장애발생 시 시도총괄관리자가 대응할 수 있도록 기능 개선

    • 이를 통해 사용자의 업무 혼선 방지 및 기록물 관리의 일관성 유지에 기여
    내부메일 가로/세로 모드 적용
    • 메일 작성 환경설정 기능 제공: 개인환경설정 내 메일관리(메모관리 하단)에 설정 항목 추가
    • 주요 설정 항목:
      • 본문 크기 (가로/세로 모드 선택 가능)
      • 기본 수신자 표시 방식
      • 최근 수신자 자동 표시 여부
    • 설정된 환경에 따라 메일 작성 화면에서 본문 크기, 수신자, 최근 수신자 정보가 자동 반영
    • 메일 상세 보기 화면에도 설정된 본문 크기가 적용되어 가독성 개선
    • 단, 타 업무에서 호출되는 메일 작성 팝업 화면은 기존 가로 모드로 고정
    • 사용자별 작성 환경 최적화를 통해 UI/UX 편의성 향상 및 작성 효율성 증대

    [문서등록대장] 문서번호 조회 및 복사 기능 변경
    • '문서등록대장' 화면 상단 문서번호 검색조건 사용 시 문서번호, 부서-문서번호가 일치한 문서만 검색 가능하게 개선
    • '문서등록대장'의 문서관리카드 내 문서번호복사 기능 사용 시 접수문서의 경우 발신기관문서번호가 복사되게 개선
    • '문서등록대장'의 문서관리카드 내 문서번호복사 기능 사용 시 기안일자가 아닌 결재완료일자가 복사되게 개선
    [문서등록대장] 접수문서 이송, 반송 처리 시 첨부된 원본문서의 관인 삭제
    • '문서등록대장' 화면에서 완료처리된 접수문서를 이송, 반송 시 원본문서의 관인 삭제 후 원본의 ODT 파일이 첨부되도록 개선
    [문서등록대장] 조회기간 설정 기능 제공
    • '문서등록대장' 화면의 등록일자 조회기간을 학교/학교 외로 세분화하여 시도설정으로 조정할 수 있도록 개선
    • 서비스공통 > 포털관리 > 운영환경관리 > 문서환경설정
      • 문서등록대장 조회기간 설정 : 학교, 학교(하위부서포함), 학교 외, 학교 외(하위부서포함)의 조회기간 각각 설정
    • 문서 사용량 증가에 따른 지연 발생 시 조회기간을 조정하여 일시적으로 대용량 조회를 줄이도록 함
    • 학교/학교 외(교육(지원)청/직속기관) 하위부서포함 조회 시 문서 건 수의 차이가 크므로, 기관 특성에 맞는 조회기간을 적용하기 위해 분리 설정 기능을 제공
    메모 작성 화면 UI 최적화
    • 메모 작성 시 가독성과 사용성을 개선하기 위해 K-에듀파인에서 제공하는 기본 비율을 적용하고, 불필요한 스크롤이 발생하지 않도록 UI를 최적화함
    • 가로 모드와 세로 모드를 선택할 수 있도록 개선하여, 사용자별로 편리한 방식으로 본문을 확인할 수 있도록 함
    • 서식 선택이 보다 유연하게 이루어질 수 있도록 상단에 서식 선택 기능을 추가하여 빠르게 적용할 수 있도록 함
    • 사용자 환경설정 기능 추가
    • 본문 모드(가로/세로) 선택 가능
    • 서식 선택 펼치기 옵션 추가
    • 자주 쓰는 과제카드나 수신자를 기본으로 선택할 수 있는 기능 추가
    • 세로 모드에서는 본문이 아래로 크게 펼쳐져 보다 넓은 화면에서 편리하게 작성할 수 있도록 함
    일정현황 조회 시 부서별 정렬 기능 추가
    • 일정현황 목록을 조회할 때 부서별로 정렬할 수 있는 기능을 추가하여 가독성과 편의성을 향상함
    • 사용자가 조회된 내용을 직접 정렬할 수 있도록 기능을 추가하여 보다 효율적으로 일정 관리가 가능하도록 함
    기록물이관 열람범위 목록을 공통 코드로 통합하여 일관성 유지
    • 열람범위 목록을 공통 코드로 등록하여 체계적으로 유지관리할 수 있도록 개선함

    2월 12일

    [문서등록대장] 비전자문서 수정 기능 확대
    • '문서등록대장' 화면에서 등록된 비전자문서에 한해 수정 가능 개선
    • 1월 23일 개선된 ‘처리완료된 비전자문서 수정할 수 없도록 개선’ 관련한 추가 배포

    2월 6일

    [문서등록대장(다년도검색)] 문서함 상단 검색 조건 확대
    • '문서등록대장(다년도검색)' 화면에서 검색조건을 기존 8개에서 15개 항목으로 확대하여 다양한 검색조건을 활용할 수 있도록 개선
    • 검색조건 항목 : 제목, 문서번호, 발신기관명, 대국민공개여부[전체/공개/부분공개/비공개], 소속부서, 처리현황[종료/발송대기/발송/추가재발송/발송보류/접수/진행/반송/이송/경유], 결재유형[전체/생산/접수], 재정문서유무[전체/재정/일반/연계], 등록일자, 기안(접수)자, 결재(담당)자, 시행종류[대외시행/대내시행/내부결재], 과제카드, 열람제한[전체/열람제한/열람제한 제외], 문서종류[전체/전자/비전자]

    1월 23일

    [문서등록대장] 처리완료된 비전자문서는 수정할 수 없도록 개선
    • '문서등록대장' 화면에서 비전자문서(생산, 접수)의 경우 문서관리카드 내 수정버튼이 비활성화 되도록 개선
    • 단, 기존 사용가능한 한건수정, 일괄수정 기능은 그대로 유지
      • 한건수정 가능항목 : *과제카드명, 관련정보, *대국민공개여부, *직원열람범위, *직원열람제한, 쪽수
      • 일괄수정 가능항목 : *대국민공개여부, *직원열람범위, *직원열람제한
    • 비전자수정이력은 2025.01.24일 등록된 비전자문서부터 보이지 않도록 개선
    [공용서식] 행정표준코드가 없는 문서 생성 방지를 위한 프로세스 개선
    • '공용서식' 화면에서 행정표준코드가 없는 문서 생성 방지를 위해 과제카드 선택 시 문서함을 보유하지 않은 조직임을 안내문구로 팝업하여 문서처리가 되지 않도록 개선
    • 서비스공통 > 조직및사용자 > (업무)조직관리 > 조직관리 : 조직정보 수정 시 행정표준코드가 없는 경우 문서함보유여부가 '아니요'로 자동설정되게 개선
    • 국가기록원에서 고시한 기록관리시스템 데이터연계 기술규격에 따라 기록물이관 시, 행정표준코드가 필수 값에 해당
    [접수대기] 스토리지 용량 최적화를 위한 접수문서 본문 PDF 파일 미생성
    • '접수대기' 화면에서 접수문서 처리 시 접수문서의 본문 PDF 파일이 생성되지 않도록 개선
    • 문서목록의 파일저장 사용 또는 문서함 하단 본문/붙임파일(탭)에서 PDF 다운로드 등 PDF 파일이 필요한 경우 즉시 변환되어 사용자에게 제공
    [문서등록대장(다년도검색)] 조회 항목과 동일하게 엑셀다운로드 기능 제공
    • '문서등록대장(다년도검색)' 화면에서 엑셀 다운로드 시 조회 항목(등록일자, 게시여부, 공개구분, 공개제한근거, 최초발송일자)과 동일하게 출력되도록 개선
    • CSV다운(기본) 항목 : 문서번호, 결재유형, 제목, 기안(접수), 결재권자, 등록일자, 수신(발신), 처리상태, 게시여부, 연계유형, 공개구분, 공개제한근거, 최초발송일자
    • CSV다운(과제정보추가) 항목 : 문서번호, 결재유형, 제목, 기안(접수), 결재권자, 등록일자, 수신(발신), 처리상태, 게시여부, 연계유형, 전자비전자, 공개제한근거, 첨부파일여부, 쪽수, 목록공개여부, 과제명, 과제카드명, 보존기한, 최초발송일자
    과제카드함 파일 다운로드 시 비전자 건 수 추가
    • 과제카드함에서 파일 다운로드 시 제공되는 항목에 비전자 건 수를 추가하여 정보 제공 범위를 확대함
    • 기존에는 과제, 과제카드, 과제담당자 등의 정보만 제공되었으나, 비전자 기록물의 관리 강화를 위해 비전자 건 수와 권수 항목을 추가
    단위과제추진현황 조회 속도 개선
    • 기존에는 시도총괄관리자(업무) 권한을 가진 사용자가 단위과제추진현황을 조회할 경우, 해당 시도의 모든 실적이 생산년도별로 한꺼번에 조회되어 시스템 부하 발생
    • 이를 방지하기 위해 선택된 부서의 정보만 조회하도록 변경하여 불필요한 데이터 조회를 줄이고 속도를 개선
    종료된 (구)업무관리 과제카드 참여자 정보 조회 가능
    • (구)업무관리 시스템에서 생성된 종료된 과제카드의 경우 참여자 정보를 확인할 수 없었음
    • 특히, 예산 정보를 확인할 때 참여자 정보가 필요하지만, K-에듀파인 과제카드가 아니어서 입력이 불가능
    • 이를 해결하기 위해 과제카드함 화면에서 종료된 과제카드를 조회한 후, 담당자 변경 기능을 통해 참여자 정보를 수정할 수 있도록 개선
    메모 조회 시 서식 적용 및 조회 속도 개선
    • 메모 작성자가 입력한 본문이 수신자에게도 동일한 형태로 유지될 수 있도록 개선함
    • 기존에는 수신자가 메모를 조회할 때 본문이 다르게 표시되는 경우가 있었으나, 이를 방지하여 일관된 서식 제공
    • 메모 작성 화면에서 최근 수신자 목록을 조회하는 속도를 최적화하여 보다 빠르게 불러올 수 있도록 함
    게시물 등록 시 첨부파일 용량 제한 적용방법 보완
    • 그룹게시물 등록 시 첨부파일 최대 용량을 선택된 게시판 중 가장 작은 값으로 적용
    • 게시판에서 허용된 용량을 초과한 파일이 등록되지 않도록 시스템의 일관성을 유지
    기록물이관 파일 생성 시 용량 확인 개선
    • 기록물이관 파일을 생성할 때 데이터베이스(DB)와 실제 서버의 저장 공간을 함께 확인하도록 기능을 추가

    1월 6일

    [문서환경설정] 전자문서유통의 처리 결과가 0건인 경우 내부메일로 알람
    • '문서환경설정' 화면에서 유통문서 수신결과가 0건인 경우 설정에 따라 내부메일이 발송되도록 개선
    • 서비스공통 > 포털관리 > 운영환경관리 > 문서환경설정 : 유통문서 수신 결과 메일설정
      • 시간 설정 : 오전 9시부터 오후 6시까지 1시간 단위로 메일 수신 시간 체크박스로 설정(다중선택 가능)
      • 수신자 설정 : 메일 받을 수신자를 조직도에서 선택하여 추가 설정(다중선택 가능)
