    2025년 문서관리 기능개선 내역

    K-에듀파인 문서관리 업무의 기능개선 사항을 배포일자별로 세부사항을 제공합니다. 주요 기능 및 화면변경에 대해서만 작성하였고 오류수정 등은 포함하지 않습니다.

    10월 28일

    [미완료문서] 미접수 문서 조회 및 일괄접수 기능 추가

    • 미완료문서 화면에서 조회유형을 미접수로 선택 후 조회 시 미접수 상태의 문서를 선택하여 일괄접수할 수 있는 기능을 추가
    • 일괄접수 기능 추가를 통해 접수대기 화면으로 이동하지 않고 바로 접수가 가능하여 업무 편의성 향상

    [통합검색] 휴직자 권한 보유자가 통합 검색 기능 사용 시 대국민공개여부 공개 문서만 통합 검색이 가능하도록 기능개선

    • 휴직자 권한 보유자가 통합검색 기능 이용 시 대국민공개여부가 공개인 문서를 조회할 수 있도록 기능 개선
    • 기존에는 휴직자가 문서를 확인할 수 없었으나, 공개 문서에 한해 통합검색을 통한 조회가 가능해져 문서 확인 편의성 향상
    [문서등록대장(다년도검색)] 키워드 검색방법 안내 팝업 추가
    • 문서등록대장(다년도검색) 화면에서 검색 시 사용 가능한 키워드 항목을 안내하는 팝업을 추가하여 사용자가 검색 조건을 쉽게 파악하고 활용할 수 있도록 개선
    • 제목 조건 오른쪽에 검색도움말 버튼을 추가하여 키워드 검색 방법을 안내하고 검색 편의성을 향상
    • 키워드 항목 : AND 검색, OR 검색, 복합 검색, NOT 검색, 순서 검색 1, 순서 검색 2

    9월 23일

    [개인수신그룹관리] 대내수신자설정여부가 아니요 인 기관을 개인수신그룹으로 등록 시 대내조직 등록 금지

    • 수신그룹관리, 개인수신그룹관리 화면에서 엑셀 파일을 통한 개인수신그룹 일괄등록 시 대내수신자설정여부가 아니오로 설정된 조직은 미등록
    • 수신그룹관리, 개인수신그룹관리에 엑셀을 업로드할 경우 대내수신자설정여부 확인 후 등록되도록 개선하여 발송처리 시 수신실패가 발생하지 않도록 변경
    [문서등록대장] 문서관리카드 내 수정 기능 추가
    • 문서등록대장 화면에서 문서관리카드 클릭 시 상단 수정 버튼을 추가하여 문서 내용을 확인한 후 바로 수정할 수 있도록 개선
    • 기존 문서등록대장 목록에서 제공하던 한건수정과 동일 기능
      • 수정 가능항목 : 과제카드명, 관련정보, 대국민공개여부, 직원열람범위, 직원열람제한, 쪽수
    [문서진행] 문서 재작성 처리 시 결재선에 공석이 존재할 경우 안내문구 추가
    • 문서진행 화면에서 문서 체크 후 상단 재작성 클릭 시 결재경로설정 팝업과 함께 공석관련 안내문구를 추가하여 공석자가 포함된 결재선을 변경하여 처리할 수 있도록 안내
    • 생상문서는 반려, 회수만 재작성 가능
    • 안내문구 : 결재경로에 공석(공석사유)이 포함되어 있습니다. 사용자의 공석정보를 확인 후 결재경로를 변경하여 처리하여 주시기 바랍니다.
    [공용서식] 문서관리카드 내 공람 인원수 표기 추가
    • 공용서식, 결재대기, 문서진행, 문서등록대장 등의 화면에서 문서관리카드 조회 시 공람탭에 인원수를 표기하여 공람여부를 바로 알 수 있도록 개선
    • 문서관리카드를 확인 할 수 있는 모든 문서함 적용
    [문서등록대장] 문서관리카드 내 문서번호복사 기능 사용 시 날짜 표기형식 개선
    • '문서등록대장' 화면의 문서관리카드 내 문서번호복사 기능 사용 시 날짜 표기형식을 개편된 행정업무편람에 맞추어 있도록 개선
    • 표기형식 : 2025. 9. 9 (O) / 2025. 09. 09 (X)
    [공용서식] 공문게시지정 시 주제영역의 전체 선택항목 삭제
    • 공용서식, 결재대기, 접수대기 등의 화면에서 문서처리 시 공문게시지정 기능 사용 중 주제 영역의 전체 선택항목 삭제

    8월 26일

    [발송처리] 수신자숨김의 경우 발송처리에서 수신자 항목이 마스킹 되도록 개선
    • '발송처리' 화면에서 마스킹된 항목을 볼 수 있는 사용자라도 수신자 항목은 항상 마스킹되도록 개선
    [재정기안] 재정문서 중 학교(지출품의)의 경우 대국민공개여부의 초기값을 설정하지 않아 사용자가 선택할 수 있도록 개선
    • 재정문서의 대국민공개여부 초기값은 '비공개'로 설정되어 있었지만 학교(지출품의)에 한해 초기값을 지정하지 않고 사용자가 직접 선택할 수 있도록 개선

    [발송대기] 문서함보유여부와 대내수신자설정여부 '아니오' 설정 시 문서 발송처리되지 않고 수신실패 되도록 개선

    • '발송대기' 화면에서 문서 발송처리 시 수신처 중 문서함보유여부와 대내수신자설정여부에 따라 수신실패 되도록 개선
    • 수신실패 조건 : 문서함보유여부와 대내수신자설정여부 중 하나라도 '아니오'면 수신실패
    • 국립학교 등 문서함보유여부와 대내수신자설정여부가 정상적으로 설정되어 있는지 점검 필요
    [배부대기] 문서 배부처리 시 조직트리에 문서함보유여부와 대내수신자설정여부를 확인하여 생성하도록 개선
    • '배부대기' 화면에서 문서 배부처리 시 배부처 지정 과정에서 문서함보유여부와 대내수신자설정여부 설정에 따라 배부처 조회 가능
    • 배부처에 조회되는 조건 : 문서함보유여부와 대내수신자설정여부 중 하나라도 '아니오'면 조회 안됨
      • 예외 조건 : 문서함보유여부와 대내수신자설정여부가 '예'로 설정된 하위부서가 있는 경우 조회 됨
    [문서진행] 국민신문고 문서를 수정 후 기안 시 회수하면 초기 문서가 아닌 수정된 문서로 반영되게 개선
    • '문서진행' 화면에서 국민신문고 결재문서를 수정 후 회수할 경우, 수정내용이 정상적으로 반영되도록 개선

    7월 22일

    [감사담당자열람이력] 조회 목록과 엑셀다운로드의 결과 값이 일치하도록 개선
    • '감사담당자열람이력' 화면에서 엑셀다운로드 시 상단 조회조건에 추가된 감사열람조직정보를 매핑하여 목록과 동일하게 출력되도록 개선
    • 엑셀다운로드 항목 : 전자/비전자, 문서번호, 문서제목, 대국민공개여부, 직원열람제한, 열람구분, 열람일시, 열람자, 열람자조직, 열람정보
    [접수대기] 일괄접수 팝업에서 접수버튼 하단에 나의결재선 기능이 위치하도록 개선
    • '접수대기' 화면의 일괄접수, 일괄담당지정 팝업 상단 신규 추가된 나의결재선 기능의 위치를 접수버튼 하단으로 변경하여 기존 접수버튼의 위치가 익숙한 사용자들의 혼란을 최소화 하도록 개선
    • 일괄접수 팝업 : 접수, 결재경로 버튼 하단에 나의결재선 위치
    • 일괄담당지정 팝업 : 결재경로 버튼 하단에 나의결재선 위치

    6월 24일

    5월 27일

    [문서환경설정] 문서환경설정에 개인정보필터 사용여부에 사용안함 설정 추가 개선
    • '문서환경설정' 화면에 사용안함 항목을 추가하여 개인정보필터 장애발생 시 시도총괄관리자가 대응할 수 있도록 기능 개선
    • 개인정보필터 서버의 장애 등 문서를 작성할 수 없는 경우 일시적으로 개인정보필터를 제한
    [문서환경설정] 개인정보필터 설정에 따라 문서 결재처리 시 개인정보필터 처리결과 안내
    • '문서환경설정' 화면에 자동(발송문서) 항목 추가. 자동 또는 자동(발송문서)로 설정한 경우 다음결재자에게 개인정보필터 처리결과를 팝업으로 안내 하도록 개선
    • 결재처리를 위한 문서관리카드 조회 시 팝업으로 개인정보필터 처리결과를 안내함으로 개인정보 유출 방지 강화
    [외부공문대기함] 재발송요청 다건처리 기능 추가
    • '외부공문대기함' 화면에서 여러 건의 재발송 문서를 한번에 처리할 수 있도록 일괄처리 기능 개선
    [외부공문처리현황] 문서함의 수신처목록 팝업에서 처리자 정보 표기 추가
    • '외부공문처리현황' 화면의 수신처수 클릭 시 팝업되는 수신처목록에서 처리자(외부공문관리담당자) 이름(ID) 추가로 표시 되도록 개선
    [외부공문관리통계현황] 통계항목에 공문게시(요청) 추가
    • '외부공문관리통계현황' 화면에 외부공문총괄관리자가 공문게시로 요청한 공문게시(요청) 건수 항목 추가 개선
    • 공문게시 주제영역을 통계현황과 분리하여 하단 별도의 표로 구성
    [외부공문처리현황] 문서함 상단 검색조건에 처리유형 추가
    • '외부공문처리현황' 화면의 검색조건에 처리유형 조건을 추가하여 공문게시, 공문처리를 구분하여 조회할 수 있도록 개선
    [외부공문대기함] 문서함 상단 검색조건에 수신일시 추가
    • '외부공문대기함' 화면의 검색조건에 수신일시 조건을 추가하여 대기중인 외부공문을 기간별로 조회할 수 있도록 개선
    [접수대기] 일괄접수, 일괄담당지정 처리 시 처리순번 제공
    • '접수대기' 화면에서 일괄접수, 일괄담당지정 버튼을 통해 문서처리 시 처리순번을 안내 하도록 개선
    • 예) 5개의 문서중 1번째
    [문서등록대장] 문서의 공개제한기간 만료 시 직원열람제한도 동일하게 변경되도록 개선
    • 문서관리카드 작성 시 설정한 공개제한기간이 만료되어 부분공개 또는 비공개 문서가 공개로 전환될 떄 직원열람제한도 설정안함으로 변경되도록 개선
    • 공개제한기간 설정 조건
      • 대국민공개여부 : 부분공개(5호) 또는 비공개(5호)
      • 공개제한사유 : 내부검토/의사결정
      • 공개제한기간 : 지정날짜까지
    [문서등록대장] 문서의 부분공개 처리에 따른 정보공개포털 열람제한 관련 안내문구 추가
